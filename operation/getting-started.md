# 🚀 시작 가이드

## ✅ 초기 설정 체크리스트

| 순서 | 항목 | 필수 |
|:---:|-----|:---:|
| 1 | 비밀번호 변경 | ✅ |
| 2 | 병원 정보 등록 | ✅ |
| 3 | 의료진 등록 | ✅ |
| 4 | 운영 시간 설정 | ✅ |
| 5 | 이벤트 등록 | ✅ |
| 6 | 예약 일정 관리 | ✅ |
| 7 | 알림 설정 | 선택 |

---

## Step 1. 비밀번호 변경

1. 파트너센터 로그인
2. 우측 상단 **프로필** 클릭
3. **설정** → **계정 관리**
4. **비밀번호 변경**
5. 새 비밀번호 입력 → **저장**

> 8자 이상, 영문+숫자+특수문자 포함

---

## Step 2. 병원 정보 등록

1. **설정** → **병원 정보**
2. 정보 입력
3. **저장**

| 항목 | 예시 |
|-----|-----|
| 병원명 | 자신있나비뇨기과의원 |
| 주소 | 서울시 강남구 테헤란로 123 |
| 전화번호 | 02-1234-5678 |
| 병원 소개 | 비뇨기과 수술 전문 |

---

## Step 3. 의료진 등록

1. **설정** → **의료진 관리** → **추가**
2. 정보 입력
3. **저장**

---

## Step 4. 운영 시간 설정

1. **설정** → **운영 시간**
2. 요일별 시간 입력
3. **저장**

---

## Step 5. 이벤트 등록

1. **이벤트 관리** → **새 이벤트**
2. 수술 정보 입력
3. 이미지 업로드
4. **등록** → 검수

👉 [이벤트 등록 자세히](../event/create.md)

---

## Step 6. 예약 일정 관리

1. **예약 관리** → **예약 일정 관리**
2. **새로 만들기**
3. 시간/인원 설정
4. 이벤트 연결
5. **저장**

👉 [예약 일정 관리 자세히](../reservation/schedule.md)

---

## 🎉 설정 완료!

모든 설정이 완료되면 앱에 병원이 노출됩니다.
