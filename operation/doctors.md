# 👨‍⚕️ 의료진 관리

## 📝 의료진 추가

1. **설정** → **의료진 관리** → **추가**
2. 정보 입력
3. **저장**

| 항목 | 필수 | 예시 |
|-----|:---:|-----|
| 이름 | ✅ | 홍길동 |
| 직함 | ✅ | 대표원장 |
| 전문 분야 | ✅ | 정관수술, 남성확대 |
| 사진 | 선택 | - |
| 학력 | 선택 | 서울대 의대 |
| 경력 | 선택 | 15년 |

---

## 📸 프로필 사진

| 항목 | 권장 |
|-----|-----|
| 크기 | 400 x 400px |
| 비율 | 1:1 |
| 용량 | 2MB 이하 |

**팁:**
- 단색 배경
- 정면 촬영
- 진료복 착용

---

## ✏️ 수정/삭제

1. **의료진 관리** → 선택
2. 수정 또는 **삭제**
