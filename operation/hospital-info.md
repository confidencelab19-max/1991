# 🏥 병원 정보 관리

## 📝 등록 항목

### 필수 정보
| 항목 | 예시 |
|-----|-----|
| 병원명 | 자신있나비뇨기과의원 |
| 주소 | 서울시 강남구 테헤란로 123 |
| 전화번호 | 02-1234-5678 |
| 병원 소개 | 비뇨기과 수술 전문 |

### 선택 정보
| 항목 | 설명 |
|-----|-----|
| 오시는 길 | 교통 안내 |
| 주차 안내 | 주차 방법 |
| 병원 사진 | 최대 10장 |

---

## ✏️ 수정 방법

1. **설정** → **병원 정보**
2. 정보 수정
3. **저장**

> ⚠️ 병원명/주소 변경 시 서류 확인 필요
