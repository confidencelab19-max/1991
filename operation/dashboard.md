# 📊 대시보드 사용법

## 대시보드 개요

대시보드에서 병원 운영 현황을 한눈에 확인하세요.

---

## 📈 구성 요소

### 1. 오늘의 현황
| 항목 | 설명 |
|-----|-----|
| 오늘 예약 | 확정된 예약 수 |
| 대기 중 | 확정 대기 예약 |
| 신규 상담 | 오늘 채팅 상담 |
| 오늘 매출 | 결제 완료 금액 |

### 2. 주요 지표
| 항목 | 설명 |
|-----|-----|
| 이번 달 예약 | 월간 예약 수 |
| 이번 달 매출 | 월간 결제 금액 |
| 예약 전환율 | 조회→예약 비율 |
| 평균 별점 | 후기 평균 |

### 3. 최근 활동
- 최근 예약 5건
- 최근 상담 5건
- 최근 후기 5건

---

## 📊 기간별 조회

1. 상단 **기간 선택** 클릭
2. 오늘/이번 주/이번 달/직접 입력

---

## ⬇️ 데이터 다운로드

통계 영역의 **다운로드** 아이콘 → Excel 파일
