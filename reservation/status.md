# 📊 예약 현황

---

## 📋 예약 현황 보기

<details>
<summary><strong>캘린더 뷰</strong></summary>

**확인 경로:**
파트너센터 → 예약관리 → 예약현황

**캘린더 표시:**
- 일별/주별/월별 보기
- 예약 상태별 색상 구분
- 담당 의사별 필터

**색상 구분:**
| 색상 | 상태 |
|------|------|
| 🟢 녹색 | 확정 |
| 🟡 노란색 | 대기 중 |
| 🔴 빨간색 | 취소 |
| ⚫ 회색 | 완료 |

</details>

<details>
<summary><strong>목록 뷰</strong></summary>

**표시 항목:**
- 예약 일시
- 고객명
- 시술명
- 담당 의사
- 예약 상태

**필터 옵션:**
- 기간별
- 상태별
- 담당 의사별
- 시술별

</details>

---

## 📈 예약 통계

<details>
<summary><strong>예약 현황 통계</strong></summary>

**확인 항목:**
- 일별/주별/월별 예약 수
- 예약 전환율
- 취소율
- 노쇼율

**분석 활용:**
- 예약이 많은 시간대 파악
- 인기 시술 분석
- 노쇼 패턴 분석

</details>
