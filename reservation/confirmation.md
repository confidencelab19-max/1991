# 예약 확정 방식

<figure><img src="../.gitbook/assets/reservation-confirm-setting.png" alt="확정 방식 설정"><figcaption>확정 방식 설정 화면</figcaption></figure>

## 확정 방식이란?

환자가 예약을 신청했을 때, 예약이 확정되는 방식을 설정합니다.

---

## 확정 방식 종류

### 수동 확정 (권장)

| 항목 | 내용 |
|------|------|
| 방식 | 병원에서 확인 후 직접 확정 |
| 흐름 | 예약 신청 → 병원 확인 → 확정/거절 → 환자 알림 |
| 장점 | 일정 조율 가능, 오버부킹 방지 |
| 단점 | 확정까지 시간 소요 |

적합한 경우: 예약 전 일정 조율이 필요한 경우, 특정 조건 확인 후 예약을 받는 경우, 인력이 충분해 빠른 확정이 가능한 경우

### 자동 확정

| 항목 | 내용 |
|------|------|
| 방식 | 예약 신청 즉시 자동 확정 |
| 흐름 | 예약 신청 → 즉시 확정 → 환자 알림 |
| 장점 | 환자 편의성 높음, 즉시 확정 |
| 단점 | 오버부킹 가능성, 일정 조율 어려움 |

적합한 경우: 예약 즉시 확정해도 문제없는 경우, 시간대별 인원 설정이 정확한 경우, 빠른 확정을 원하는 경우

---

## 설정 방법

파트너센터 로그인 후 좌측 메뉴에서 예약 관리를 클릭하고, 상단 설정 탭을 클릭합니다. 확정 방식 항목에서 원하는 방식을 선택합니다.

---

## 주의사항

자동 확정 사용 시 시간대별 인원 설정을 정확히 해주세요. 설정된 인원을 초과하면 예약이 불가능하므로, 여유 있게 설정하는 것이 좋습니다.

처음 앱예약을 사용하신다면 수동 확정으로 시작하시는 것을 권장합니다. 운영이 익숙해지면 자동 확정으로 전환할 수 있습니다.

---

## 확정 방식 비교

| 항목 | 수동 확정 | 자동 확정 |
|------|-----------|-----------|
| 확정 속도 | 병원 확인 후 | 즉시 |
| 일정 조율 | 가능 | 어려움 |
| 오버부킹 위험 | 낮음 | 있음 |
| 운영 편의성 | 확인 필요 | 자동 처리 |
| 환자 편의성 | 대기 필요 | 즉시 확정 |

---

## 자주 묻는 질문

### Q. 확정 방식을 언제든 변경할 수 있나요?

네, 언제든지 변경할 수 있습니다. 단, 변경 전에 신청된 예약은 기존 방식이 적용됩니다.

### Q. 수동 확정 시 얼마 안에 확정해야 하나요?

가능한 빨리 확정해주시는 것이 좋습니다. 24시간 내 확정을 권장하며, 오래 대기하면 환자가 취소할 수 있습니다.
