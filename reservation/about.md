# 📱 앱예약 소개

{% hint style="success" %}
**앱예약으로 예약 관리를 자동화하세요!**

전화/카톡 없이 고객이 앱에서 직접 예약할 수 있습니다.
{% endhint %}

---

## 📋 앱예약이란?

<details>
<summary><strong>서비스 개요</strong></summary>

앱예약은 고객이 자신있나 앱에서 직접 예약하는 기능입니다.

**장점:**
- 24시간 예약 접수 가능
- 전화 응대 업무 감소
- 예약 현황 자동 관리
- 노쇼 방지 알림 자동 발송

**고객 혜택:**
- 원하는 시간에 바로 예약
- 대기 시간 없음
- 예약 확인/변경 편리

</details>

---

## ⚙️ 예약 확정 방식

<details>
<summary><strong>자동 확정</strong></summary>

고객이 예약하면 자동으로 확정됩니다.

**장점:**
- 빠른 예약 확정
- 응대 업무 감소
- 고객 만족도 향상

**주의사항:**
- 예약 일정 관리 필수
- 중복 예약 방지 설정 필요

**권장 대상:**
- 예약 가능 시간이 명확한 병원
- 담당자 상주가 어려운 병원

</details>

<details>
<summary><strong>수동 확정</strong></summary>

고객 예약 후 병원에서 확정/거절을 결정합니다.

**장점:**
- 예약 검토 후 확정
- 일정 조율 가능

**주의사항:**
- 빠른 확정 필요 (1시간 내 권장)
- 미확정 시 고객 이탈 가능

**권장 대상:**
- 예약 전 상담이 필요한 경우
- 일정 조율이 필요한 경우

</details>
