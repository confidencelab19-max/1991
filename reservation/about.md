# 앱예약이란

<figure><img src="../.gitbook/assets/reservation-patient-flow.png" alt="앱예약 흐름"><figcaption>환자의 예약 흐름</figcaption></figure>

## 앱예약 소개

앱예약은 환자가 자신있나 앱을 통해 24시간 언제든지 병원 방문 예약을 할 수 있는 기능입니다. 전화 없이도 간편하게 예약할 수 있어 환자 편의성이 높아지고, 병원은 예약 전화 응대 업무를 줄일 수 있습니다.

---

## 환자의 예약 흐름

환자는 이벤트를 확인하고 예약하기를 클릭한 후, 날짜를 선택하고 시간을 선택하면 예약이 완료됩니다.

---

## 병원의 예약 관리 흐름

새 예약 알림을 받으면 예약을 확인하고 확정 또는 거절을 처리합니다. 처리 후 환자에게 알림이 자동 발송됩니다.

---

## 앱예약의 장점

### 환자 입장

| 장점 | 설명 |
|------|------|
| 24시간 예약 | 언제든지 예약 가능 |
| 간편한 절차 | 전화 없이 몇 번의 터치로 예약 |
| 예약 확인 | 앱에서 예약 내역 확인 |
| 리마인드 알림 | 예약 전 알림으로 일정 관리 |

### 병원 입장

| 장점 | 설명 |
|------|------|
| 전화 응대 감소 | 예약 전화 업무 감소 |
| 효율적 일정 관리 | 예약 현황 한눈에 파악 |
| 노쇼 감소 | 자동 리마인드로 노쇼율 감소 |
| 데이터 축적 | 예약 데이터로 운영 개선 |

---

## 예약 설정 항목

| 설정 항목 | 설명 |
|-----------|------|
| 예약 가능 시간 | 예약을 받을 요일과 시간대 |
| 예약 간격 | 예약 시간 간격 (30분/1시간) |
| 시간대별 인원 | 같은 시간에 받을 최대 예약 수 |
| 예약 가능 기간 | 며칠 후까지 예약 가능한지 |
| 당일 예약 마감 | 당일 예약 마감 시간 |
| 확정 방식 | 자동 확정 / 수동 확정 |

## 예약 알림

새 예약이 들어오면 아래 채널로 알림을 받을 수 있습니다.

| 알림 채널 | 설명 |
|-----------|------|
| 앱 푸시 | 파트너센터 앱으로 알림 |
| 이메일 | 등록된 이메일로 발송 |
| 알림톡 | 카카오 알림톡으로 발송 |

## 예약 상태

| 상태 | 설명 |
|------|------|
| 대기 | 예약 요청됨, 확정 대기 중 |
| 확정 | 예약 확정됨 |
| 취소 | 환자 또는 병원이 취소 |

## 자주 묻는 질문

### Q. 앱예약을 사용하지 않을 수도 있나요?

네, 앱예약 기능을 OFF로 설정하면 환자는 전화나 채팅으로만 상담/예약이 가능합니다.

### Q. 예약과 상담의 차이가 뭔가요?

예약은 특정 날짜/시간에 병원 방문 예약이고, 상담은 전화나 채팅으로 문의하는 것입니다. 예약 없이 상담만 진행한 후 병원에서 직접 예약을 잡을 수도 있습니다.
