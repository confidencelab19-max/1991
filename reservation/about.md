# 📅 앱예약이란?

## 앱예약 소개

고객이 앱에서 직접 수술 상담 예약을 하는 기능입니다.

---

## 🎯 장점

| 장점 | 설명 |
|-----|-----|
| 24시간 예약 | 영업시간 외에도 접수 |
| 전화 감소 | 전화 응대 업무 절감 |
| 노쇼 감소 | 알림톡 리마인드 |
| 효율적 관리 | 시간대별 인원 조절 |

---

## 🔄 예약 흐름

### 고객
이벤트 확인 → 예약하기 → 날짜/시간 선택 → 예약 신청 → 확정 알림


### 병원
예약 알림 → 파트너센터 확인 → 확정/거절 → 고객 알림
