## 앱예약의 장점

{% columns %}
{% column %}
{% hint style="success" %}

영업시간 외에도 예약 접수가 가능합니다.
{% endhint %}
{% endcolumn %}

{% column %}
{% hint style="info" %}
**전화 응대 감소**

{% endcolumn %}

{% endhint %}
{% endcolumn %}

## 빠른 설정

앱예약을 시작하려면 3가지만 설정하면 됩니다.

{% stepper %}
{% step %}
### 1) 예약 가능 시간 설정

예약을 받을 요일/시간을 먼저 정합니다.
{% endstep %}

{% step %}
### 2) 예약 확정 방식 선택

자동 확정 또는 수동 확정을 선택합니다.
{% endstep %}

{% step %}
### 3) 알림 설정

새 예약을 놓치지 않게 알림을 켭니다.
{% endstep %}
{% endstepper %}
