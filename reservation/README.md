## 앱예약의 장점

| 장점 | 설명 |
|------|------|
| 24시간 예약 | 영업시간 외에도 예약 접수 가능 |
| 전화 응대 감소 | 예약 전화 응대 업무 감소 |
| 노쇼 감소 | 알림톡으로 예약 리마인드 |
| 효율적 일정 관리 | 예약 현황 한눈에 파악 |

## 빠른 설정

앱예약을 시작하려면 3가지만 설정하면 됩니다.

| 단계 | 설정 내용 |
|------|-----------|
| 1 | 예약 가능 시간 설정 |
| 2 | 예약 확정 방식 선택 (자동/수동) |
| 3 | 알림 설정 |
