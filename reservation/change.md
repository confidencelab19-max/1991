# 예약 변경/취소

## 예약 변경

확정된 예약의 날짜나 시간을 변경할 수 있습니다.

### 변경 가능 항목

| 항목 | 변경 가능 |
|------|-----------|
| 예약 날짜 | 가능 |
| 예약 시간 | 가능 |
| 이벤트 | 불가 (취소 후 재예약) |

### 변경 방법

예약 목록에서 변경할 예약을 클릭하고 수정 버튼을 클릭합니다. 날짜/시간을 변경한 후 저장 버튼을 클릭하면 환자에게 변경 알림이 자동 발송됩니다.

---

## 예약 취소

### 병원에서 취소하는 경우

예약 목록에서 취소할 예약을 클릭하고 취소 버튼을 클릭합니다. 취소 사유를 입력하고 확인을 클릭하면 환자에게 취소 알림이 자동 발송됩니다.

### 환자가 취소하는 경우

환자가 앱에서 직접 취소할 수 있습니다. 취소 시 병원에 알림이 발송되며, 파트너센터에서 취소된 예약을 확인할 수 있습니다.

---

## 취소 사유 관리

취소 사유를 기록하면 취소 패턴을 분석하고 개선할 수 있습니다.

### 주요 취소 사유

| 사유 | 설명 |
|------|------|
| 환자 요청 | 환자가 직접 취소 요청 |
| 일정 변경 | 병원 일정으로 인한 취소 |
| 연락 불가 | 환자 연락이 안 됨 |
| 기타 | 기타 사유 |

---

## 노쇼 처리

예약 시간에 환자가 방문하지 않은 경우 노쇼 처리합니다. 예약 목록에서 해당 예약을 클릭하고 노쇼 버튼을 클릭합니다. 노쇼 내역은 노쇼 관리에서 확인할 수 있습니다.

---

## 자주 묻는 질문

### Q. 환자가 예약 당일에 취소하면 어떻게 되나요?

예약 당일 취소도 가능합니다. 잦은 당일 취소가 문제라면 예약 정책을 안내하는 것이 좋습니다.

### Q. 취소된 예약을 복구할 수 있나요?

취소된 예약은 복구되지 않습니다. 필요한 경우 새로 예약을 받아주세요.
