# 📋 새 예약 확인하기

## 🔔 알림 수신

| 채널 | 설명 |
|-----|-----|
| 앱 푸시 | 파트너 앱 |
| 이메일 | 등록된 이메일 |
| SMS | 등록된 번호 |

---

## 📋 예약 상태

| 상태 | 설명 |
|-----|-----|
| 🟡 대기 | 확정 대기 |
| 🟢 확정 | 예약 확정 |
| 🔴 취소 | 취소됨 |
| ⚫ 노쇼 | 미방문 |
| ✅ 완료 | 방문 완료 |

---

## ✅ 확정하기

1. **예약 관리** → **예약 목록**
2. 대기 예약 선택
3. **확정** 클릭
4. 고객에게 알림 발송

---

## ❌ 거절하기

1. 대기 예약 선택
2. **거절** 클릭
3. 사유 선택
4. **확인** → 알림 발송
