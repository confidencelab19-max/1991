# 새 예약 확인하기

<figure><img src="../.gitbook/assets/reservation-list.png" alt="예약 목록"><figcaption>예약 목록 화면</figcaption></figure>

## 새 예약 알림

새로운 예약이 들어오면 아래 채널로 알림을 받습니다.

| 채널 | 설명 |
|------|------|
| 앱 푸시 | 파트너센터 앱 알림 |
| 이메일 | 등록된 이메일로 발송 |
| 알림톡 | 카카오 알림톡 (설정 시) |

---

## 예약 목록 확인

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 예약 관리를 클릭합니다.

### 예약 목록 화면

| 항목 | 설명 |
|------|------|
| 예약일시 | 예약된 날짜와 시간 |
| 환자명 | 예약한 환자 이름 |
| 연락처 | 환자 연락처 |
| 이벤트 | 예약한 이벤트명 |
| 상태 | 대기/확정/취소/완료 |

### 필터 기능

| 필터 | 설명 |
|------|------|
| 날짜 | 특정 기간 예약 조회 |
| 상태 | 상태별 필터 (대기/확정 등) |
| 검색 | 환자명/연락처로 검색 |

---

## 예약 확정하기

수동 확정 방식인 경우, 새 예약을 확인하고 확정해야 합니다.

### 확정 방법

예약 목록에서 대기 상태인 예약을 클릭하고, 예약 상세 정보를 확인한 후 확정 버튼을 클릭합니다. 환자에게 확정 알림이 자동 발송됩니다.

### 거절 방법

예약을 받을 수 없는 경우, 예약 목록에서 해당 예약을 클릭하고 거절 버튼을 클릭합니다. 거절 사유를 입력하면 환자에게 거절 알림이 자동 발송됩니다.

잦은 거절은 환자 경험을 저하시킬 수 있습니다. 예약 설정(가능 시간, 인원)을 적절히 조절해주세요.

---

## 예약 상태 설명

| 상태 | 설명 | 다음 액션 |
|------|------|-----------|
| 대기 | 예약 신청됨, 확정 대기 | 확정 또는 거절 |
| 확정 | 예약 확정됨 | 변경/취소 가능 |
| 취소 | 취소된 예약 | - |
| 노쇼 | 미방문 처리 | - |
| 완료 | 방문 완료 | - |

---

## 알림 설정

알림을 받을 채널을 설정하세요. 파트너센터 우측 상단 설정 아이콘을 클릭하고 알림 설정 메뉴에서 원하는 채널을 ON/OFF합니다.

---

## 자주 묻는 질문

### Q. 예약 알림을 못 받았어요

알림 설정을 확인해주세요. 앱 푸시는 알림 권한 허용 여부를 확인하고, 이메일은 스팸함과 이메일 주소를 확인하세요. 알림톡은 설정 여부를 확인해주세요.

### Q. 오래된 예약을 한 번에 확인하고 싶어요

예약 목록에서 날짜 필터를 사용하면 특정 기간의 예약을 조회할 수 있습니다.
