# ⚙️ 예약 설정

{% hint style="info" %}
**예약 설정을 정확히 해야 원활한 예약 관리가 가능합니다.**
{% endhint %}

---

## 📋 기본 설정

<details>
<summary><strong>예약 확정 방식</strong></summary>

**설정 경로:**
파트너센터 → 예약관리 → 예약설정

| 방식 | 설명 |
|------|------|
| 자동 확정 | 예약 즉시 자동 확정 |
| 수동 확정 | 병원에서 확정/거절 결정 |

**수동 확정 시 주의:**
- 미확정 상태로 24시간 경과 시 자동 취소
- 빠른 확정 권장 (1시간 내)

</details>

<details>
<summary><strong>예약 가능 시간 설정</strong></summary>

**설정 항목:**
- 예약 시작 시간
- 예약 종료 시간
- 점심시간 제외
- 시간 단위 (30분/1시간)

**예시 설정:**
예약 가능 시간: 09:00 ~ 17:00 점심시간: 12:00 ~ 13:00 (예약 불가) 시간 단위: 30분 마지막 예약: 17:00 (시술 시간 고려)
</details>

<details>
<summary><strong>동시 예약 제한</strong></summary>

같은 시간대에 받을 수 있는 최대 예약 수

**설정 예시:**
| 시간대 | 최대 예약 |
|--------|----------|
| 09:00-10:00 | 3건 |
| 10:00-11:00 | 3건 |
| ... | ... |

**권장 설정:**
- 의사 1인당 시간당 2-3건
- 시술 종류별 소요 시간 고려

</details>

<details>
<summary><strong>예약 마감 시간</strong></summary>

예약 신청 마감 기준

**옵션:**
- 당일 예약 가능
- 1일 전까지 예약
- 2일 전까지 예약
- 3일 전까지 예약

**권장:**
- 일반 시술: 당일 또는 1일 전
- 수술: 2-3일 전

</details>

## 📅 예약 일정 관리

<details>
<summary><strong>일정 만들기</strong></summary>

특정 기간의 예약 일정을 설정합니다.

**설정 방법:**
1. 파트너센터 → 예약관리 → 예약일정
2. "일정 추가" 클릭
3. 기간, 시간, 담당의 설정
4. 저장

</details>

<details>
<summary><strong>일정 복사</strong></summary>

기존 일정을 복사하여 새 기간에 적용

**사용 예시:**
- 이번 주 일정을 다음 주에 복사
- 특정 요일 일정을 다른 요일에 적용

</details>

<details>
<summary><strong>특정 시간 예약 차단</strong></summary>

특정 시간대만 예약을 받지 않을 때

**사용 예시:**
- 수요일 오후 회의로 예약 차단
- 특정 날짜 오전만 예약 가능

**설정 방법:**
1. 예약일정에서 해당 시간 선택
2. "예약 차단" 설정
3. 차단 사유 입력 (선택)

</details>
