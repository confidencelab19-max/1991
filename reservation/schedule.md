# ⚙️ 예약 일정 관리 만들기

## 예약 일정 관리란?

여러 이벤트의 예약 설정을 그룹으로 관리합니다.

---

## ⚙️ 설정 항목

| 항목 | 설명 | 예시 |
|-----|-----|-----|
| 이름 | 구분용 | 일반 상담 예약 |
| 예약 가능 시간 | 요일별 | 월~금 10-19시 |
| 점심시간 | 예약 불가 | 12-13시 |
| 예약 간격 | 시간 단위 | 1시간 |
| 시간대별 인원 | 동시 예약 | 3명 |
| 예약 가능 기간 | 며칠 후까지 | 30일 |
| 당일 마감 | 당일 예약 | 2시간 전 |
| 확정 방식 | 확정 방법 | 즉시 확정 |

---

## 📝 만들기

### Step 1
**예약 관리** → **예약 일정 관리** → **새로 만들기**

### Step 2. 기본 정보
| 항목 | 예시 |
|-----|-----|
| 이름 | 평일 상담 예약 |
| 설명 | 월-금 일반 상담 |

### Step 3. 시간 설정
| 항목 | 예시 |
|-----|-----|
| 예약 가능 시간 | 10:00-19:00 |
| 점심시간 | 12:00-13:00 |
| 예약 간격 | 1시간 |

### Step 4. 인원/기간
| 항목 | 예시 |
|-----|-----|
| 시간대별 인원 | 3명 |
| 예약 가능 기간 | 30일 |
| 당일 마감 | 2시간 전 |

### Step 5. 확정 방식
- 병원 확인 후 확정
- 즉시 확정

### Step 6
**저장**

---

## 🔗 이벤트 연결

1. **이벤트 관리** → 이벤트 선택
2. **예약 설정** 탭
3. 예약 일정 관리 선택
4. **저장**
