# 예약 일정 설정

<figure><img src="../.gitbook/assets/reservation-schedule-setting.png" alt="예약 일정 설정"><figcaption>예약 일정 설정 화면</figcaption></figure>

## 예약 일정 설정이란?

환자가 예약할 수 있는 시간을 설정합니다. 병원 운영 시간에 맞게 예약 가능 시간을 설정해주세요.

---

## 설정 방법

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 예약 관리를 클릭하고, 상단 설정 탭을 클릭합니다.

### 설정 항목

| 항목 | 설명 | 예시 |
|------|------|------|
| 예약 가능 요일 | 예약을 받을 요일 선택 | 월~금 |
| 시작 시간 | 예약 시작 시간 | 09:00 |
| 종료 시간 | 예약 종료 시간 | 18:00 |
| 점심 시간 | 점심 시간 제외 | 12:00~13:00 |
| 예약 간격 | 예약 시간 간격 | 30분 |
| 시간대별 인원 | 동시간대 최대 예약 수 | 2명 |
| 예약 가능 기간 | 최대 며칠 후까지 | 30일 |
| 당일 마감 시간 | 당일 예약 마감 | 2시간 전 |

---

## 상세 설정 가이드

### 예약 가능 요일

예약을 받을 요일을 선택합니다. 휴진일은 선택 해제합니다.

### 시작/종료 시간

| 설정 | 권장 |
|------|------|
| 시작 시간 | 병원 오픈 시간 |
| 종료 시간 | 마지막 예약 가능 시간 |

마감 1시간 전까지만 예약을 받으면 여유 있게 마감할 수 있습니다.

### 점심 시간

점심 시간을 설정하면 해당 시간에는 예약이 불가능합니다. 점심 시간이 없으면 설정하지 않아도 됩니다.

### 예약 간격

| 간격 | 적합한 경우 |
|------|-------------|
| 30분 | 짧은 상담, 간단한 시술 |
| 1시간 | 충분한 상담 시간 필요 |

### 시간대별 인원

같은 시간에 받을 수 있는 최대 예약 수입니다. 의사가 여러 명이거나 동시 상담이 가능한 경우 늘릴 수 있습니다.

### 예약 가능 기간

| 기간 | 설명 |
|------|------|
| 7일 | 일주일 후까지 예약 가능 |
| 14일 | 2주 후까지 예약 가능 |
| 30일 | 한 달 후까지 예약 가능 |

### 당일 마감 시간

예약 시간 몇 시간 전까지 예약을 받을지 설정합니다. 예를 들어 "2시간 전" 설정 시, 오후 3시 예약은 오후 1시까지만 가능합니다.

## 저장

모든 설정 후 저장 버튼을 클릭합니다. 저장 즉시 앱에 반영됩니다.

## 자주 묻는 질문

### Q. 요일별로 다른 시간을 설정할 수 있나요?

네, 요일별로 다른 운영 시간을 설정할 수 있습니다. 각 요일을 선택하고 개별적으로 시간을 설정해주세요.

### Q. 특정 날짜만 예약을 막고 싶어요

휴진일 설정에서 특정 날짜를 휴진일로 등록하면 해당 날짜는 예약이 불가능합니다. 휴진일 설정 가이드를 참고해주세요.
