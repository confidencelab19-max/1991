# ✅ 예약 확정하기

{% hint style="warning" %}
**예약은 빠르게 확정해주세요!**

미확정 상태가 길어지면 고객이 이탈할 수 있습니다.
{% endhint %}

---

## 📋 예약 확정 방법

<details>
<summary><strong>새 예약 확인</strong></summary>

**알림 수신:**
- 앱 푸시 알림
- 이메일 알림
- SMS 알림 (설정 시)

**확인 경로:**
파트너센터 → 예약관리 → 새 예약

</details>

<details>
<summary><strong>예약 확정</strong></summary>

**확정 방법:**
1. 새 예약 목록에서 예약 선택
2. 예약 상세 정보 확인
3. "예약 확정" 버튼 클릭

**확정 시 자동 처리:**
- 고객에게 확정 알림 발송
- 예약 현황에 반영
- 캘린더에 자동 등록

</details>

<details>
<summary><strong>예약 거절</strong></summary>

**거절 사유:**
- 해당 시간 예약 불가
- 시술 불가 사유
- 기타

**거절 방법:**
1. 예약 선택
2. "예약 거절" 버튼 클릭
3. 거절 사유 선택
4. 고객에게 거절 알림 발송

**주의:**
- 거절 시 고객에게 친절한 안내 필수
- 대안 시간 제안 권장

</details>

## ⏱️ 확정 시간 가이드

{% hint style="danger" %}
**미확정 상태로 24시간 경과 시 자동 취소됩니다.**
{% endhint %}

**권장 확정 시간:**
| 시간 | 평가 |
|------|------|
| 10분 이내 | ⭐⭐⭐⭐⭐ 최고 |
| 1시간 이내 | ⭐⭐⭐⭐ 우수 |
| 3시간 이내 | ⭐⭐⭐ 보통 |
| 12시간 이내 | ⭐⭐ 개선 필요 |
| 24시간 초과 | 자동 취소 |
