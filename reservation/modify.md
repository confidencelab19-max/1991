# 🔄 예약 변경/취소

---

## 📋 예약 변경

<details>
<summary><strong>일정 변경</strong></summary>

**변경 가능 항목:**
- 예약 날짜
- 예약 시간
- 담당 의사

**변경 방법:**
1. 예약 현황에서 해당 예약 선택
2. "예약 변경" 클릭
3. 새로운 일정 선택
4. 변경 확정

**변경 시 자동 처리:**
- 고객에게 변경 알림 발송
- 기존 일정 자동 해제
- 새 일정에 반영

</details>

<details>
<summary><strong>고객 요청 변경</strong></summary>

고객이 앱에서 변경 요청 시

**처리 방법:**
1. 변경 요청 알림 확인
2. 요청 내용 검토
3. 승인 또는 거절

</details>

---

## ❌ 예약 취소

<details>
<summary><strong>병원 측 취소</strong></summary>

**취소 사유:**
- 불가피한 휴진
- 의료진 사정
- 기타 사유

**취소 방법:**
1. 예약 선택
2. "예약 취소" 클릭
3. 취소 사유 입력
4. 취소 확정

**취소 시 주의:**
- 고객에게 취소 알림 발송
- 진심 어린 사과 메시지 권장
- 대안 일정 제안

</details>

<details>
<summary><strong>고객 측 취소</strong></summary>

고객이 앱에서 취소 시

**처리:**
- 자동으로 예약 취소 처리
- 병원에 취소 알림 발송
- 해당 시간 예약 가능으로 전환

**취소 시점별 정책:**
| 시점 | 정책 |
|------|------|
| 3일 전 | 자유 취소 |
| 1-2일 전 | 취소 가능 (경고) |
| 당일 | 노쇼 카운트 가능 |

</details>
