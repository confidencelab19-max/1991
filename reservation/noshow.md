# 노쇼 관리

## 노쇼란?

노쇼(No-Show)는 예약한 환자가 예약 시간에 방문하지 않는 경우입니다.

---

## 노쇼 현황 확인

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 예약 관리를 클릭하고, 상태 필터에서 노쇼를 선택합니다.

### 확인 가능 정보

| 항목 | 설명 |
|------|------|
| 예약일시 | 노쇼된 예약 날짜/시간 |
| 환자명 | 노쇼 환자 이름 |
| 이벤트 | 예약했던 이벤트 |
| 노쇼 처리일 | 노쇼로 처리한 날짜 |

---

## 노쇼 처리 방법

예약 시간이 지났는데 환자가 방문하지 않은 경우, 예약 목록에서 해당 예약을 선택하고 노쇼 버튼을 클릭합니다.

---

## 노쇼율 관리

### 노쇼율 확인

대시보드 또는 통계에서 노쇼율을 확인할 수 있습니다.

### 노쇼 줄이는 방법

| 방법 | 설명 |
|------|------|
| 리마인드 알림 | 예약 전날 알림톡 자동 발송 |
| 예약 확인 전화 | 예약 전날 확인 전화 |
| 노쇼 정책 안내 | 예약 시 노쇼 정책 사전 안내 |

자신있나에서는 예약 전날 환자에게 자동으로 리마인드 알림을 발송합니다.

---

## 자주 묻는 질문

### Q. 노쇼율이 높으면 어떻게 되나요?

노쇼율이 높으면 병원 운영 효율이 떨어집니다. 리마인드 알림, 예약 확인 전화 등을 활용해보세요.

### Q. 노쇼 환자에게 불이익을 줄 수 있나요?

현재 플랫폼 차원의 노쇼 패널티 기능은 없습니다. 병원 자체적으로 노쇼 정책을 안내하실 수 있습니다.
