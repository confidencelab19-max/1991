# 🚫 노쇼 관리

{% hint style="danger" %}
**노쇼는 병원 운영에 큰 손실을 초래합니다.**

체계적인 노쇼 관리로 손실을 최소화하세요.
{% endhint %}

---

## 📋 노쇼 정의

<details>
<summary><strong>노쇼 기준</strong></summary>

**노쇼 인정 기준:**
- 예약 시간 경과 후 연락 없이 미방문
- 예약 당일 취소 (사전 연락 없음)

**노쇼 제외 기준:**
- 사전 연락 후 취소 (1일 전)
- 불가피한 사유 (긴급 상황 등)
- 병원 측 사유로 인한 취소

</details>

---

## 🛡️ 노쇼 예방

<details>
<summary><strong>리마인드 알림</strong></summary>

**자동 알림 설정:**
| 시점 | 알림 내용 |
|------|----------|
| 2일 전 | 예약 확인 알림 |
| 1일 전 | 방문 준비 안내 |
| 당일 아침 | 최종 리마인드 |

**설정 경로:**
파트너센터 → 예약관리 → 알림설정

</details>

<details>
<summary><strong>예약 확인 전화</strong></summary>

중요 예약은 직접 확인 전화를 권장합니다.

**확인 전화 대상:**
- 고가 시술 예약
- 첫 방문 고객
- 노쇼 이력 고객

**통화 스크립트:**
"안녕하세요, 자신있나 OOO병원입니다. 내일 예약하신 OOO 시술 확인차 연락드렸습니다. 예정대로 방문 가능하신가요?"
</details>

## ⚠️ 노쇼 대응

<details>
<summary><strong>노쇼 처리</strong></summary>

**처리 방법:**
1. 예약 현황에서 해당 예약 선택
2. "노쇼 처리" 클릭
3. 노쇼 사유 기록

**노쇼 처리 시:**
- 고객 노쇼 이력에 기록
- 향후 예약 시 노쇼 이력 표시

</details>

<details>
<summary><strong>노쇼 패널티</strong></summary>

**플랫폼 패널티:**
| 노쇼 횟수 | 패널티 |
|----------|--------|
| 1회 | 경고 |
| 2회 | 예약 시 알림 표시 |
| 3회 이상 | 예약 제한 (7일) |

**병원 자체 정책:**
- 노쇼 고객 예약 제한
- 예약금 제도 운영 (선택)

</details>

<details>
<summary><strong>노쇼 고객 관리</strong></summary>

**노쇼 이력 확인:**
파트너센터 → 예약관리 → 고객 정보 → 노쇼 이력

**재예약 시 대응:**
- 노쇼 이력 고객 표시
- 예약 확인 전화 필수
- 당일 예약 제한 고려

</details>
