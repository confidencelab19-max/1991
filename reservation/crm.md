# 🔗 CRM 연동

**기존 CRM과 연동하여 효율적으로 관리하세요!**
{% endhint %}

---

## 📋 베가스 CRM 연동

<details>
<summary><strong>연동 방법</strong></summary>

**Step 1: API 키 발급**
1. 베가스 CRM 관리자 페이지 접속
2. 설정 → API 관리
3. "새 API 키 발급" 클릭
4. 키 복사

**Step 2: 파트너센터 연동**
1. 파트너센터 → 설정 → CRM연동
2. "베가스 CRM" 선택
3. API 키 입력
4. 연동 테스트
5. 연동 완료

</details>

<details>
<summary><strong>연동 항목</strong></summary>

**자동 동기화 항목:**
- 예약 정보
- 고객 정보
- 시술 이력
- 상담 내역

**동기화 주기:**
- 실시간 동기화
- 또는 5분 단위 동기화 (설정 가능)

</details>
