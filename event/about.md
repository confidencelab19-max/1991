# 🎉 이벤트란?

## 이벤트 소개

**이벤트**는 자신있나 앱에 노출되는 **수술 상품**입니다.

수술 정보와 가격을 등록하면 고객에게 노출되어 예약/상담으로 이어집니다.

---

## 🎯 이벤트의 역할

| 역할 | 설명 |
|-----|-----|
| 수술 정보 전달 | 수술 방법, 장점 안내 |
| 가격 안내 | 투명한 가격 공개 |
| 예약/상담 유도 | 고객 액션 유도 |

---

## 📋 이벤트 구성 요소

| 요소 | 예시 |
|-----|-----|
| 수술명 | 정관수술 원데이 |
| 카테고리 | 정관수술 |
| 가격 | 350,000원 |
| 상세 설명 | 당일수술, 당일퇴원 |
| 이미지 | 수술 관련 이미지 |

---

## 💡 좋은 이벤트 조건

1. **명확한 제목**: "정관수술 원데이 (당일퇴원)"
2. **경쟁력 있는 가격**
3. **상세한 설명**
4. **고화질 이미지**

---

## 📍 노출 위치

- 병원 상세 페이지
- 카테고리 검색
- 메인 추천 (광고)
