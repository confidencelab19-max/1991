# 이벤트란

## 이벤트 개념

이벤트는 자신있나 앱에서 환자에게 노출되는 수술 상품입니다. 병원에서 제공하는 수술의 정보, 가격, 담당 의사 등을 이벤트로 등록하면, 환자들이 이를 보고 상담을 신청하거나 예약을 할 수 있습니다.

## 이벤트 구성 요소

| 요소 | 설명 | 필수 |
|------|------|------|
| 수술 종류 | 수술 카테고리 (정관수술, 포경수술 등) | 필수 |
| 이벤트명 | 이벤트 제목 | 필수 |
| 가격 | 수술 가격 (VAT 포함) | 필수 |
| 이벤트 설명 | 수술 상세 설명 | 필수 |
| 이벤트 이미지 | 대표 이미지 | 필수 |
| 담당 의사 | 수술 담당 의사 | 필수 |
| 이벤트 기간 | 시작일, 종료일 | 선택 |

| 카테고리 | 설명 |
|----------|------|
| 정관수술 | 남성 영구 피임 수술 |
| 포경수술 | 성인/소아 포경수술 |
| 정관복원술 | 정관수술 복원 |
| 발기부전 | 발기부전 수술적 치료 |
| 전립선 | 전립선비대증 등 수술 |
| 남성확대 | 음경확대, 귀두확대 등 |
| 조루수술 | 조루 개선 수술 |
| 요로결석 | 요로결석 제거 수술 |

## 이벤트와 수익

### 수익 발생 구조

환자가 이벤트를 보고 상담을 시작하면, 상담 건당 수수료가 발생합니다.

### 수수료 안내

| 항목 | 내용 |
|------|------|
| 과금 기준 | 상담 시작 건당 |
| 수수료 | 건당 일정 금액 (별도 안내) |

수수료 정책은 별도 안내 자료를 참고해주세요. 문의는 partner@jasininna.com으로 해주세요.

## 자주 묻는 질문

### Q. 이벤트를 몇 개까지 등록할 수 있나요?

등록 개수에 제한이 없습니다. 병원에서 제공하는 모든 수술을 이벤트로 등록할 수 있습니다.

### Q. 같은 수술을 여러 이벤트로 등록할 수 있나요?

네, 같은 수술이라도 다른 조건(가격, 담당 의사 등)으로 여러 이벤트를 등록할 수 있습니다.

### Q. 이벤트 기간을 설정하지 않으면 어떻게 되나요?

기간을 설정하지 않으면 수동으로 종료하기 전까지 계속 노출됩니다.
