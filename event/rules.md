{% endhint %}

---

## 📜 기본 운영 규칙

<details>
<summary><strong>1. 가격 정책</strong></summary>

**기본 원칙:**
- 모든 가격은 VAT 포함 총액으로 표시
- 정상가는 실제 시술 정가 기준
- 이벤트가는 정상가 대비 할인된 가격

**가격 표시 규정:**
- 할인율: 정상가 대비 최대 50%까지 권장
- 추가 비용: 별도 발생 시 명시 필수
- 패키지: 포함 항목 상세 명시

**금지 사항:**
- ❌ 허위 정상가 표시 (뻥튀기 가격)
- ❌ 미끼 가격 (추가 비용 숨김)
- ❌ 가격 외 조건 숨김

</details>

<details>
<summary><strong>2. 이벤트 기간</strong></summary>

**기간 설정:**
- 최소 기간: 7일
- 최대 기간: 90일
- 연장: 종료 전 재등록 필요

**상시 이벤트:**
- 종료일 미지정 시 상시 이벤트로 등록
- 단, 3개월마다 갱신 필요

**조기 종료:**
- 언제든 종료 가능
- 기존 예약 건은 이벤트가 적용

</details>

<details>
<summary><strong>3. 노출 순위</strong></summary>

**순위 결정 요소:**
| 요소 | 비중 |
|------|------|
| 광고비 입찰가 | 40% |
| 전환율 (예약/조회) | 25% |
| 리뷰 평점 | 20% |
| 콘텐츠 품질 | 15% |

**순위 올리는 방법:**
- 부가광고 이용
- 전환율 개선 (콘텐츠 품질 향상)
- 리뷰 관리
- 전후사진 등록

➡️ [광고 관리 가이드](../ads/)

</details>

<details>
<summary><strong>4. 이벤트 수정</strong></summary>

**수정 가능 항목:**
- 이벤트명
- 가격 (인상/인하)
- 설명 내용
- 이미지

**수정 시 재검수:**
- 의료법 관련 내용 수정 시 재검수 필요
- 단순 오타 수정은 재검수 불필요

**주의사항:**
- 가격 인상 시 기존 예약은 기존가 적용
- 시술 내용 변경은 신규 이벤트로 등록

</details>

| 5단계 | 파트너 계약 해지 | 영구 |

</details>

<details>
<summary><strong>주요 제재 사유</strong></summary>

**즉시 중지:**
- 의료법 중대 위반
- 허위 정보 등록
- 타 병원 비방

**경고 후 중지:**
- 경미한 의료법 위반
- 가격 정보 오류
- 이미지 저작권 위반

</details>
