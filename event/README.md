## 이벤트 등록 흐름

<figure><img src="../.gitbook/assets/event-review-process.png" alt="이벤트 등록 흐름"><figcaption>이벤트 등록부터 노출까지의 흐름</figcaption></figure>

## 이벤트 등록 팁

### 검수 빠르게 통과하는 팁

의료법 위반 문구를 사용하지 않고, 가격은 VAT 포함 금액으로 명시하세요. 명확하고 구체적인 설명을 작성하고, 고화질 이미지를 사용하면 검수가 빠르게 통과됩니다.

### 주의사항

허위/과장 광고, 타 병원 비방, 최저가 보장 문구 사용은 금지됩니다.
