## 🎯 이벤트 운영 핵심 포인트

<details>
<summary><strong>1. 매력적인 이벤트 구성</strong></summary>

**필수 요소:**
- 명확한 시술명
- 구체적인 가격 정보
- 시술 전/후 사진 (있는 경우)
- 상세한 시술 설명

**전환율 높이는 팁:**
- 합리적인 할인율 (10-30% 권장)
- 명확한 혜택 제시
- 신뢰감 있는 정보 제공

</details>

<details>
<summary><strong>2. 의료법 준수</strong></summary>

모든 이벤트는 의료법을 준수해야 합니다.

**주요 체크 포인트:**
- 금지 표현 사용 여부
- 전문의 자격 표기 정확성
- 전후사진 규정 준수
- 가격 표시 규정 준수

➡️ [의료법 가이드 바로가기](../medical-law/)

</details>

<details>
<summary><strong>3. 검수 통과 노하우</strong></summary>

**반려율 낮추는 방법:**
- 등록 전 의료법 가이드 확인
- 금지 표현 자가 점검
- 이미지 내 텍스트 확인
- 가격 정보 정확성 확인

➡️ [검수 프로세스 보기](review-process.md)
➡️ [반려 사유 및 수정 방법](rejected.md)

</details>
