## 이벤트 등록 흐름

| 단계     | 설명          | 소요 시간   |
| ------ | ----------- | ------- |
| 이벤트 작성 | 수술 정보 입력    | 10분     |
| 검수 요청  | 등록 완료 버튼 클릭 | 즉시      |
| 검수 진행  | 내부 검수팀 검토   | 1-2 영업일 |
| 승인/반려  | 결과 통보       | -       |
| 앱 노출   | 승인 시 자동 노출  | 즉시      |

## 이벤트 등록 팁

{% hint style="success" %}
**검수 빠르게 통과하는 팁**\n\n의료법 위반 문구를 피하세요.\n가격은 VAT 포함 금액으로 명시하세요.\n설명은 구체적으로 쓰고, 고화질 이미지를 사용하세요.
{% endhint %}

**주의사항**\n\n허위/과장 광고, 타 병원 비방, 최저가 보장 문구는 금지됩니다.
{% endhint %}
