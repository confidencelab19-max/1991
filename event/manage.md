# 📋 이벤트 관리

## 이벤트 상태

| 상태 | 설명 |
|-----|-----|
| 검수 중 | 검수 진행 중 |
| 운영 중 | 정상 노출 |
| 반려 | 검수 탈락 (수정 필요) |
| 종료 | 노출 중단 |

---

## 노출 순서 변경

1. **이벤트 관리** → **노출 순서**
2. 드래그로 순서 변경
3. **저장**

---

## 일시 비노출

1. **이벤트 관리** → 이벤트 선택
2. **노출** 토글 OFF
3. 자동 저장

---

## 이벤트 종료

1. 이벤트 선택 → **종료**
2. 앱에서 즉시 비노출
3. 기존 예약은 유지

---

## 종료 이벤트 재등록

1. **종료된 이벤트** 탭
2. 이벤트 선택 → **재등록**
