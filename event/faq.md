# ❓ 이벤트 FAQ

---

## 📋 자주 묻는 질문

<details>
<summary><strong>Q. 이벤트 검수는 얼마나 걸리나요?</strong></summary>

**A.** 평일 기준 1-2 영업일 내 검수가 완료됩니다.

- 평일 접수: 1-2 영업일
- 주말/공휴일 접수: 다음 영업일부터 처리
- 재검수: 당일 처리 우선

검수 현황은 파트너센터 → 이벤트관리에서 확인 가능합니다.

</details>

<details>
<summary><strong>Q. 반려된 이벤트는 어떻게 수정하나요?</strong></summary>

**A.** 반려 사유 확인 후 수정하여 재검수 요청하세요.

1. 파트너센터 → 이벤트관리 → 반려된 이벤트 클릭
2. "반려 사유" 확인
3. "수정하기" 클릭 → 해당 내용 수정
4. "재검수 요청" 클릭

➡️ [반려 사유별 수정 가이드](rejected.md)

</details>

<details>
<summary><strong>Q. 이벤트 가격을 변경할 수 있나요?</strong></summary>

**A.** 네, 언제든 변경 가능합니다.

**주의사항:**
- 가격 인상 시: 기존 예약은 기존가 적용
- 가격 인하 시: 즉시 적용
- 대폭 변경 시: 재검수 필요할 수 있음

</details>

<details>
<summary><strong>Q. 이벤트 노출 순위는 어떻게 결정되나요?</strong></summary>

**A.** 복합적인 요소로 결정됩니다.

| 요소 | 비중 |
|------|------|
| 광고비 입찰가 | 40% |
| 전환율 | 25% |
| 리뷰 평점 | 20% |
| 콘텐츠 품질 | 15% |

순위를 높이려면 부가광고 이용, 콘텐츠 개선, 리뷰 관리를 권장합니다.

➡️ [광고 관리 가이드](../ads/)

</details>

<details>
<summary><strong>Q. 하나의 시술에 여러 이벤트를 등록할 수 있나요?</strong></summary>

**A.** 네, 가능합니다.

**예시:**
- 귀두필러 - 신규 고객 이벤트
- 귀두필러 - 재방문 고객 이벤트
- 귀두필러 - 패키지 이벤트

단, 동일한 내용의 중복 이벤트는 제한됩니다.

</details>

<details>
<summary><strong>Q. 이벤트 종료 후 재등록 가능한가요?</strong></summary>

**A.** 네, 언제든 재등록 가능합니다.

- 기존 이벤트 복사 기능 이용
- 가격/내용 수정 후 재등록
- 새로 검수 필요

</details>
