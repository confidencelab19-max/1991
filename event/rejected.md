# ❌ 반려 사유 및 수정

{% hint style="warning" %}
**반려된 이벤트는 사유 확인 후 수정하여 재검수 요청하세요.**
{% endhint %}

---

## 📋 주요 반려 사유

<details>
<summary><strong>1. 의료법 위반 표현</strong></summary>

**반려 사유:**
금지된 표현이 포함되어 있습니다.

**주요 금지 표현:**
- 최고, 최상, 유일, 국내 최초
- 100% 성공, 완벽한, 부작용 없는
- 연예인/유명인 언급
- 타 병원 비교

**수정 방법:**
1. 반려 사유에서 해당 표현 확인
2. 해당 표현 삭제 또는 수정
3. 재검수 요청

**수정 예시:**
- ❌ "국내 최고의 귀두확대 전문" → ✅ "10년 경력의 귀두확대 전문의"
- ❌ "100% 만족 보장" → ✅ "체계적인 사후 관리 제공"

</details>

<details>
<summary><strong>2. 이미지 내 금지 표현</strong></summary>

**반려 사유:**
이미지 내 금지된 표현이 포함되어 있습니다.

**주요 사례:**
- 이미지 내 "최저가", "파격 할인" 등 텍스트
- 금지 표현이 포함된 배너 이미지
- 타 병원 로고/이미지 포함

**수정 방법:**
1. 해당 이미지에서 금지 표현 제거
2. 새 이미지로 교체
3. 재검수 요청

</details>

<details>
<summary><strong>3. 전후사진 규정 위반</strong></summary>

**반려 사유:**
전후사진이 의료법 규정을 준수하지 않습니다.

**주요 위반 사항:**
- 환자 동의서 미첨부
- 촬영 조건 상이 (조명, 각도 등)
- 과도한 보정
- 시술 효과 과장

**수정 방법:**
1. 규정에 맞는 전후사진으로 교체
2. 환자 동의서 첨부 확인
3. 재검수 요청

➡️ [전후사진 규정 보기](../medical-law/photos.md)

</details>

<details>
<summary><strong>4. 가격 정보 오류</strong></summary>

**반려 사유:**
가격 정보가 적절하지 않습니다.

**주요 사례:**
- 정상가보다 이벤트가가 높음
- 비현실적인 할인율 (80% 이상)
- 가격 0원 표시
- VAT 미포함 표시

**수정 방법:**
1. 가격 정보 확인 및 수정
2. VAT 포함 총액으로 수정
3. 재검수 요청

</details>

<details>
<summary><strong>5. 카테고리 불일치</strong></summary>

**반려 사유:**
선택한 카테고리와 시술 내용이 일치하지 않습니다.

**수정 방법:**
1. 올바른 카테고리로 재선택
2. 시술 내용 확인
3. 재검수 요청

</details>

## 🔄 재검수 요청 방법

<details>
<summary><strong>재검수 요청 절차</strong></summary>

1. **반려 사유 확인**
   - 파트너센터 → 이벤트관리 → 반려된 이벤트 클릭
   - "반려 사유" 확인

2. **수정 진행**
   - "수정하기" 버튼 클릭
   - 반려 사유에 해당하는 내용 수정

3. **재검수 요청**
   - 수정 완료 후 "재검수 요청" 버튼 클릭
   - 재검수는 당일 처리 우선

**주의사항:**
- 반려 사유를 정확히 확인 후 수정
- 동일 사유로 3회 이상 반려 시 등록 제한 가능

</details>
