# 이벤트 반려 목록

<figure><img src="../.gitbook/assets/event-rejected-list.png" alt="반려 목록"><figcaption>이벤트 반려 목록 화면</figcaption></figure>

## 이벤트 반려란?

검수 과정에서 기준에 맞지 않는 이벤트는 반려됩니다. 반려된 이벤트는 수정 후 재검수 요청이 가능합니다.

---

## 반려 목록 확인

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 이벤트 관리를 클릭하고, 이벤트 반려 목록을 클릭합니다.

### 확인 가능 정보

| 항목 | 설명 |
|------|------|
| 이벤트명 | 반려된 이벤트 제목 |
| 반려일 | 반려 처리된 날짜 |
| 반려 사유 | 상세 반려 사유 |

---

## 주요 반려 사유

### 의료법 위반

| 사유 | 예시 |
|------|------|
| 과장 광고 | "100% 성공률", "완벽한 수술" |
| 비교 광고 | "최저가", "타 병원보다 저렴" |
| 허위 광고 | "부작용 없음", "통증 제로" |

### 정보 미흡

| 사유 | 예시 |
|------|------|
| 가격 미기재 | 가격 없이 "상담 후 결정" |
| 설명 부족 | 수술 설명이 너무 짧음 |
| 이미지 없음 | 대표 이미지 미첨부 |

### 부적절한 이미지

| 사유 | 예시 |
|------|------|
| 수술 장면 | 수술 중 사진 |
| 환부 사진 | 시술 전후 환부 노출 |
| 저작권 위반 | 타 사이트 이미지 무단 사용 |

### 기타

| 사유 | 예시 |
|------|------|
| 담당 의사 미선택 | 의사 정보 연결 안 됨 |
| 카테고리 불일치 | 수술 종류와 내용 불일치 |

---

## 재검수 요청 방법

반려 목록에서 해당 이벤트를 선택하고 반려 사유를 확인합니다. 수정 버튼을 클릭하여 반려 사유에 맞게 내용을 수정한 후 재검수 요청 버튼을 클릭합니다.

재검수도 최초 검수와 동일하게 1-2 영업일이 소요됩니다.

---

## 자주 묻는 질문

### Q. 반려 사유에 동의하지 않으면 어떻게 하나요?

반려 사유에 이의가 있으시면 partner@jasininna.com으로 문의해주세요. 담당자가 검토 후 안내드리겠습니다.

### Q. 몇 번까지 재검수 요청이 가능한가요?

횟수 제한은 없습니다. 반려 사유를 수정하여 다시 요청해주세요.
