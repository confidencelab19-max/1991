# 🔍 검수 프로세스

{% hint style="info" %}
**모든 이벤트는 의료법 준수 여부를 검수합니다.**

검수 기준을 미리 확인하여 반려를 방지하세요.
{% endhint %}

---

## 📋 검수 절차

<details>
<summary><strong>1단계: 자동 검수</strong></summary>

이벤트 등록 시 자동으로 기본 사항을 점검합니다.

**자동 검수 항목:**
- 필수 입력 항목 누락 여부
- 이미지 규격 충족 여부
- 가격 정보 유효성
- 금지 키워드 포함 여부 (1차)

**자동 반려 사유:**
- 필수 항목 미입력
- 이미지 미등록
- 가격 0원 또는 음수

</details>

<details>
<summary><strong>2단계: 담당자 검수</strong></summary>

자동 검수 통과 후 담당자가 상세 검수합니다.

**검수 항목:**
| 항목 | 검수 내용 |
|------|----------|
| 의료법 | 금지 표현, 허위/과장 광고 여부 |
| 이미지 | 저작권, 타병원 이미지 도용 여부 |
| 가격 | 정상가/이벤트가 적정성 |
| 전후사진 | 의료법 준수 여부 |
| 정보 일치 | 시술 내용과 카테고리 일치 여부 |

**검수 소요 시간:**
- 평일 접수: 1-2 영업일
- 주말/공휴일 접수: 다음 영업일부터 처리

</details>

<details>
<summary><strong>3단계: 결과 안내</strong></summary>

**승인:**
- 이벤트 즉시 앱에 노출
- 알림톡/이메일로 승인 안내

**반려:**
- 반려 사유 상세 안내
- 수정 후 재검수 요청 가능
- 재검수는 당일 처리 우선

**수정 요청:**
- 일부 수정 필요 시
- 수정 완료 후 자동 재검수

</details>

---

## ⏱️ 검수 현황 확인

<details>
<summary><strong>검수 현황 보는 방법</strong></summary>

**확인 경로:**
파트너센터 → 이벤트관리 → 이벤트 목록

**상태 구분:**
| 상태 | 설명 |
|------|------|
| 검수대기 | 검수 요청됨, 대기 중 |
| 검수중 | 담당자 검수 진행 중 |
| 승인 | 검수 완료, 앱 노출 중 |
| 반려 | 검수 불통과, 수정 필요 |
| 수정요청 | 일부 수정 후 재검수 대기 |

</details>
