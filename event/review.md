# ⚖️ 이벤트 검수 기준

## 검수 소요 시간

| 유형 | 시간 |
|-----|-----|
| 신규 등록 | 1~2 영업일 |
| 수정 재검수 | 1 영업일 |

---

## ✅ 승인 기준

- 의료법 준수
- 사실 기반 정보
- 명확한 가격 표시
- 적절한 이미지

---

## ❌ 반려 사유

| 사유 | 예시 |
|-----|-----|
| 허위 정보 | 거짓 경력, 조작 사진 |
| 과장 광고 | "100% 성공", "최고" |
| 불명확 가격 | "~부터", 추가비용 미표시 |
| 부적절 이미지 | 저화질, 저작권 위반 |
| 타병원 비방 | 경쟁병원 언급 |

---

## 반려 시 처리

1. 반려 알림 확인
2. 반려 사유 확인
3. 내용 수정
4. **재등록**
