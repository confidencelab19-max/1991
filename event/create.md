# 이벤트 등록하기

## 이벤트 등록 전 준비사항

이벤트를 등록하기 전에 아래 항목이 준비되어 있는지 확인해주세요.

| 항목 | 확인 |
|------|------|
| 병원 정보 입력 완료 | 필수 |
| 의사 정보 1명 이상 등록 | 필수 |
| 이벤트 이미지 준비 | 필수 |

---

## 이벤트 등록 단계

### Step 1: 이벤트 설정 메뉴 이동

파트너센터 로그인 후 좌측 메뉴에서 이벤트 관리를 클릭하고, 이벤트 설정을 클릭한 후 이벤트 등록 버튼을 클릭합니다.

### Step 2: 기본 정보 입력

| 항목 | 입력 내용 | 필수 |
|------|-----------|------|
| 수술 종류 | 카테고리 선택 (정관수술, 포경수술 등) | 필수 |
| 이벤트명 | 이벤트 제목 (30자 이내) | 필수 |
| 가격 | 수술 가격 (VAT 포함 금액) | 필수 |
| 담당 의사 | 수술 담당 의사 선택 | 필수 |

이벤트명 작성 팁: 수술명을 명확히 포함하고, 특징이나 장점을 간결하게 표현하세요. 예를 들어 "무도 정관수술", "당일 퇴원 포경수술" 등이 좋은 예시입니다.

### Step 3: 상세 설명 입력

| 항목 | 입력 내용 | 필수 |
|------|-----------|------|
| 이벤트 설명 | 수술 상세 설명 (1000자 이내) | 필수 |
| 수술 특징 | 주요 특징 및 장점 | 필수 |
| 주의사항 | 수술 전후 주의사항 | 권장 |
| 포함 내역 | 가격에 포함된 항목 | 권장 |

설명에 포함하면 좋은 내용: 수술 방법 설명, 수술 시간, 마취 방법, 회복 기간, 수술 후 관리

### Step 4: 이미지 업로드

| 항목 | 내용 |
|------|------|
| 이미지 수 | 최소 1장, 최대 5장 |
| 파일 형식 | JPG, PNG |
| 파일 크기 | 각 10MB 이하 |
| 권장 해상도 | 1200x800px 이상 |

이미지 주의사항: 수술 장면이나 환부 사진 사용 금지, 환자 얼굴 노출 금지, 타 병원 이미지 사용 금지, 저작권 문제 없는 이미지만 사용

### Step 5: 이벤트 기간 설정 (선택)

| 항목 | 내용 |
|------|------|
| 시작일 | 이벤트 노출 시작 날짜 |
| 종료일 | 이벤트 노출 종료 날짜 |

기간을 설정하지 않으면 수동 종료 전까지 계속 노출됩니다.

### Step 6: 검수 요청

모든 정보를 입력한 후 임시저장(나중에 이어서 작성)하거나 검수 요청(검수팀에 검토 요청)을 할 수 있습니다.

---

## 검수 프로세스

| 단계 | 내용 | 소요 시간 |
|------|------|-----------|
| 검수 요청 | 등록 완료 버튼 클릭 | 즉시 |
| 검수 진행 | 검수팀에서 내용 검토 | 1-2 영업일 |
| 결과 통보 | 승인 또는 반려 안내 | 이메일/알림 |

검수는 평일 기준 1-2 영업일이 소요됩니다. 주말/공휴일에는 검수가 진행되지 않습니다.

## 검수 통과 팁

### 피해야 할 문구

| 금지 문구 | 이유 |
|-----------|------|
| "최저가", "가장 저렴한" | 비교 광고 금지 |
| "100% 성공", "완벽한" | 과장 광고 금지 |
| "부작용 없는" | 허위 광고 금지 |
| "유명 연예인 시술" | 허위 광고 금지 |

### 포함해야 할 내용

| 필수 내용 | 설명 |
|-----------|------|
| 정확한 수술명 | 의학적으로 정확한 명칭 |
| VAT 포함 가격 | 추가 비용 여부 명시 |
| 담당 의사 정보 | 전문의 자격 명시 |

## 자주 묻는 질문

### Q. 이벤트가 반려되면 어떻게 하나요?

반려 사유를 확인하고 해당 부분을 수정한 후 다시 검수 요청하시면 됩니다. 반려 사유는 이메일과 파트너센터에서 확인할 수 있습니다.

### Q. 가격을 "상담 후 결정"으로 할 수 있나요?

아니요, 정확한 가격을 명시해야 합니다. 추가 비용이 발생할 수 있는 경우 설명란에 명시해주세요.

### Q. 임시저장한 이벤트는 어디서 확인하나요?

이벤트 설정 > 이벤트 목록에서 "작성중" 상태로 확인할 수 있습니다.
