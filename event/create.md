# ➕ 이벤트 등록

{% hint style="info" %}
**이벤트 등록은 고객 유치의 시작입니다!**

아래 가이드를 따라 매력적인 이벤트를 등록하세요.
{% endhint %}

---

## 📋 이벤트 등록 절차

<details>
<summary><strong>Step 1: 기본 정보 입력</strong></summary>

**필수 입력 항목:**

| 항목 | 설명 | 예시 |
|------|------|------|
| 시술 카테고리 | 대분류 선택 | 귀두확대 |
| 세부 시술 | 소분류 선택 | 필러확대시술 |
| 이벤트명 | 노출될 제목 | [신규 고객 특가] 귀두필러 확대 |
| 정상가 | 시술 정가 | 1,500,000원 |
| 이벤트가 | 할인 적용가 | 990,000원 |

**작성 팁:**
- 이벤트명은 시술명 + 혜택을 명확히
- 가격은 VAT 포함 총액으로 표시
- 과도한 할인율은 신뢰도 하락 요인

</details>

<details>
<summary><strong>Step 2: 상세 정보 입력</strong></summary>

**시술 설명:**
시술 방법, 소요 시간, 회복 기간 등을 상세히 작성

[권장 구성]

시술 개요 (2-3문장)
시술 방법 (단계별 설명)
시술 시간 및 회복 기간
권장 대상
주의사항
**시술 정보:**
| 항목 | 입력값 |
|------|--------|
| 시술 시간 | 30분 |
| 마취 방법 | 국소마취 |
| 회복 기간 | 3-5일 |
| 입원 여부 | 당일 퇴원 |
| 내원 횟수 | 시술 1회 + 경과 확인 1회 |

</details>

<details>
<summary><strong>Step 3: 이미지 등록</strong></summary>

**대표 이미지 (필수):**
- 크기: 720 x 720px (1:1 비율)
- 형식: JPG, PNG
- 용량: 5MB 이하

**추가 이미지 (선택):**
- 최대 5장까지 등록 가능
- 시술 과정, 시설 사진 등

**이미지 내 텍스트 주의:**
- 금지 표현 포함 시 반려
- 과도한 가격 강조 지양
- 의료법 준수 필수

</details>

<details>
<summary><strong>Step 4: 전후사진 연결 (선택)</strong></summary>

등록된 전후사진을 이벤트에 연결할 수 있습니다.

**연결 방법:**
1. "전후사진 연결" 버튼 클릭
2. 등록된 전후사진 목록에서 선택
3. 최대 5세트까지 연결 가능

**주의사항:**
- 검수 완료된 전후사진만 연결 가능
- 해당 시술과 관련된 사진만 연결

➡️ [전후사진 등록 가이드](../photos/upload.md)

</details>

<details>
<summary><strong>Step 5: 검수 요청</strong></summary>

모든 정보 입력 후 검수 요청

**검수 소요 시간:**
- 평일: 1-2 영업일
- 주말/공휴일: 다음 영업일부터 처리

**검수 결과:**
- 승인: 즉시 앱에 노출
- 반려: 반려 사유 확인 후 수정 필요

➡️ [검수 프로세스 상세 보기](review-process.md)

</details>

## ⚠️ 이벤트 등록 시 주의사항

{% hint style="danger" %}
**의료법 위반 시 이벤트가 반려됩니다!**
{% endhint %}

<details>
<summary><strong>금지 표현 체크리스트</strong></summary>

**사용 불가 표현:**
- ❌ 최고, 최상, 유일, 국내 최초
- ❌ 100% 성공, 완벽한, 부작용 없는
- ❌ 연예인 OOO 선택, 유명인 시술
- ❌ 특정 대학병원과 비교
- ❌ 검증되지 않은 효과 주장

**사용 가능 표현:**
- ✅ 10년 경력 전문의
- ✅ 000건 시술 경험
- ✅ 체계적인 사후 관리

➡️ [금지 표현 전체 목록](../medical-law/prohibited.md)

</details>
