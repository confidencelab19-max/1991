# 이벤트 수정하기

<figure><img src="../.gitbook/assets/event-edit.png" alt="이벤트 수정"><figcaption>이벤트 수정 화면</figcaption></figure>

## 이벤트 수정 안내

등록된 이벤트의 정보를 수정할 수 있습니다. 단, 일부 항목은 수정 후 재검수가 필요합니다.

---

## 수정 방법

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 이벤트 관리를 클릭하고, 이벤트 설정을 클릭합니다. 수정할 이벤트를 선택한 후 수정 버튼을 클릭합니다.

---

## 수정 가능 항목

### 즉시 반영 항목

아래 항목은 수정 후 즉시 반영됩니다.

| 항목 | 설명 |
|------|------|
| 이벤트 기간 | 시작일, 종료일 변경 |
| 이벤트 상태 | 노출 ON/OFF |

### 재검수 필요 항목

아래 항목은 수정 후 재검수가 필요합니다.

| 항목 | 설명 |
|------|------|
| 이벤트명 | 제목 변경 |
| 가격 | 가격 변경 |
| 이벤트 설명 | 상세 설명 변경 |
| 이벤트 이미지 | 이미지 변경/추가/삭제 |
| 담당 의사 | 담당 의사 변경 |
| 수술 종류 | 카테고리 변경 |

재검수가 필요한 항목을 수정하면 검수 완료 전까지 이벤트가 앱에서 노출되지 않습니다. 수정은 신중하게 진행해주세요.

---

## 이벤트 일시 중지

이벤트를 삭제하지 않고 일시적으로 노출을 중단할 수 있습니다.

### 일시 중지 방법

이벤트 목록에서 해당 이벤트를 선택하고 노출 OFF 토글을 클릭합니다.

### 노출 재개 방법

이벤트 목록에서 해당 이벤트를 선택하고 노출 ON 토글을 클릭합니다.

---

## 이벤트 삭제

삭제된 이벤트는 복구할 수 없습니다. 일시적인 중단이 필요한 경우 "노출 OFF"를 사용해주세요.

### 삭제 방법

이벤트 목록에서 해당 이벤트를 선택하고, 삭제 버튼을 클릭한 후 확인 메시지에서 삭제를 클릭합니다.

---

## 자주 묻는 질문

### Q. 가격만 수정해도 재검수가 필요한가요?

네, 가격 변경은 재검수가 필요합니다. 가격 변경이 잦다면 고객센터에 문의해주세요.

### Q. 재검수 중에 기존 이벤트는 어떻게 되나요?

재검수 중에는 기존 내용이 유지되지 않고 이벤트가 일시적으로 노출되지 않습니다.
