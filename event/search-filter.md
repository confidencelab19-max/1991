# 🔎 이벤트 검색/필터

{% hint style="info" %}
**2026년 1월 신규 기능!**

이벤트 승인요청 내역을 더 빠르고 쉽게 확인할 수 있습니다.
{% endhint %}

---

## 🔍 검색 기능

<details>
<summary><strong>이벤트명 검색</strong></summary>

**검색 방법:**
1. 이벤트 목록 상단 검색창 클릭
2. 이벤트명 또는 키워드 입력
3. 엔터 또는 검색 버튼 클릭

**검색 팁:**
- 정확한 이벤트명 입력
- 시술명으로도 검색 가능
- 부분 일치 검색 지원

</details>

## 🎛️ 필터 기능

<details>
<summary><strong>상태별 필터</strong></summary>

| 상태 | 설명 |
|------|------|
| 전체 | 모든 상태 표시 |
| 검수대기 | 검수 요청 후 대기 중 |
| 검수중 | 담당자 검수 진행 중 |
| 승인 | 검수 완료, 앱 노출 중 |
| 반려 | 검수 불통과 |
| 수정요청 | 일부 수정 필요 |
| 종료 | 이벤트 기간 종료 |

</details>

<details>
<summary><strong>기간별 필터</strong></summary>

| 기간 | 설명 |
|------|------|
| 오늘 | 오늘 등록/수정된 이벤트 |
| 최근 7일 | 최근 1주일 |
| 최근 30일 | 최근 1개월 |
| 사용자지정 | 시작일~종료일 직접 선택 |

</details>

<details>
<summary><strong>카테고리별 필터</strong></summary>

시술 대분류 기준으로 필터링

**카테고리 목록:**
- 귀두확대
- 정관수술
- 길이연장
- 대체진피
- 실리콘보형물
- 필러확대
- 자가지방확대수술
- 조루수술
- 포경수술
- 이물질제거
- 남성체형
- 모발이식
- 인조고환
- 요로결석
- 전립선수술
- 만곡증
- 발기부전수술

</details>

## 📥 엑셀 다운로드

<details>
<summary><strong>다운로드 방법</strong></summary>

1. 원하는 필터 적용
2. "엑셀 다운로드" 버튼 클릭
3. 파일 저장

**다운로드 항목:**
- 이벤트명
- 카테고리
- 정상가/이벤트가
- 등록일
- 상태
- 조회수
- 예약수

</details>
