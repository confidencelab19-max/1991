# 질문 확인하기

## 질문 확인 방법

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 의사 Q&A를 클릭합니다.

### 질문 목록 화면

| 항목 | 설명 |
|------|------|
| 질문 제목 | 환자가 올린 질문 제목 |
| 카테고리 | 수술 카테고리 |
| 작성일 | 질문 작성 날짜 |
| 답변 상태 | 답변 대기/답변 완료 |
