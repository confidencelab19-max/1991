# 📤 전후사진 등록

{% hint style="info" %}
**전후사진 등록 시 의료법 규정을 반드시 준수하세요!**
{% endhint %}

---

## 📋 등록 절차

<details>
<summary><strong>Step 1: 사진 준비</strong></summary>

**사진 규격:**
| 항목 | 규격 |
|------|------|
| 크기 | 최소 800 x 800px |
| 형식 | JPG, PNG |
| 용량 | 5MB 이하 |

**촬영 조건:**
- 전/후 동일한 조명
- 전/후 동일한 각도
- 전/후 동일한 거리
- 과도한 보정 금지

</details>

<details>
<summary><strong>Step 2: 동의서 준비</strong></summary>

**필수 동의서:**
- 전후사진 활용 동의서
- 개인정보 제공 동의서

**동의서 포함 내용:**
- 활용 목적 (온라인 광고)
- 활용 범위 (자신있나 앱/웹)
- 동의 철회 방법

</details>

<details>
<summary><strong>Step 3: 사진 업로드</strong></summary>

**업로드 경로:**
파트너센터 → 전후사진 → 사진등록

**입력 항목:**
- 시술 카테고리
- 시술명
- 시술일
- 전 사진 / 후 사진
- 환자 동의서

</details>

<details>
<summary><strong>Step 4: 검수 요청</strong></summary>

업로드 완료 후 자동으로 검수 요청됩니다.

**검수 기간:** 1-3 영업일

➡️ [검수 기준 보기](criteria.md)

</details>
