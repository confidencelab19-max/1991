# 채팅 상담

<figure><img src="../.gitbook/assets/chat-list.png" alt="채팅 상담"><figcaption>채팅 상담 목록 화면</figcaption></figure>

## 채팅 상담이란?

환자가 앱에서 병원과 1:1 채팅으로 상담할 수 있는 기능입니다. 실시간으로 환자의 질문에 답변하고 예약을 유도할 수 있습니다.

---

## 채팅 상담 확인

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 상담 관리를 클릭하고, 채팅 상담을 클릭합니다.

### 채팅 목록 화면

| 항목 | 설명 |
|------|------|
| 환자명 | 채팅 중인 환자 이름 |
| 마지막 메시지 | 최근 주고받은 메시지 |
| 시간 | 마지막 메시지 시간 |
| 상태 | 대기/진행중/완료 |

---

## 채팅 응대하기

### 새 메시지 알림

새 채팅이 들어오면 앱 푸시 알림, 이메일 알림(설정 시)으로 알림을 받습니다.

### 답변 방법

채팅 목록에서 환자를 선택하고, 채팅창에서 메시지를 확인한 후 답변을 입력하고 전송합니다.

<figure><img src="../.gitbook/assets/chat-conversation.png" alt="채팅 대화"><figcaption>채팅 대화 화면</figcaption></figure>

---

## 빠른 답변 (템플릿)

자주 사용하는 답변을 템플릿으로 등록해 빠르게 응대할 수 있습니다.

### 템플릿 예시

| 상황 | 템플릿 |
|------|--------|
| 인사 | "안녕하세요! 자신있나 OO병원입니다. 무엇을 도와드릴까요?" |
| 가격 문의 | "해당 수술의 가격은 OO만원입니다. 자세한 상담을 원하시면 예약해주세요." |
| 예약 유도 | "예약을 원하시면 앱에서 원하시는 날짜와 시간을 선택해주세요." |

---

## 채팅 상담 팁

채팅 상담 시 빠른 응대(5분 이내), 친절한 말투, 정확한 정보 제공, 예약/방문 유도를 해주세요. 장문보다는 짧고 명확하게 답변하는 것이 좋습니다.

---

## 자주 묻는 질문

### Q. 채팅 알림을 못 받았어요

알림 설정을 확인해주세요. 설정 > 알림 설정에서 채팅 알림이 켜져 있는지 확인하세요.

### Q. 채팅을 종료하고 싶어요

채팅 상세 화면에서 상담 완료 버튼을 클릭하면 해당 채팅이 완료 처리됩니다.
