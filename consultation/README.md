## 상담 채널

자신있나에서 제공하는 상담 채널입니다.

| 채널  | 설명                 |
| --- | ------------------ |
| 앱전화 | 환자가 앱에서 병원으로 전화 연결 |
| 채팅  | 앱 내 1:1 채팅 상담      |

## 상담의 중요성

{% columns %}
{% column %}
{% hint style="info" %}
**첫 인상**

상담이 환자의 첫 인상을 결정합니다.
{% endhint %}
{% endcolumn %}

{% column %}
{% hint style="success" %}
**전환율**

좋은 상담이 예약/방문으로 이어집니다.
{% endhint %}
{% endcolumn %}

{% column %}
{% hint style="warning" %}
**재방문**

친절한 응대가 재방문을 유도합니다.
{% endhint %}
{% endcolumn %}
{% endcolumns %}

## 상담 팁

{% hint style="success" %}
채팅은 **5분 이내** 답변을 권장합니다. 말투는 친절하게, 정보는 정확하게 안내하세요. 가능하면 상담을 **예약/방문**으로 자연스럽게 연결하세요.
{% endhint %}
