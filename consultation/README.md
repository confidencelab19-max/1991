## 상담 채널

자신있나에서 제공하는 상담 채널입니다.

| 채널 | 설명 |
|------|------|
| 앱전화 | 환자가 앱에서 병원으로 전화 연결 |
| 채팅 | 앱 내 1:1 채팅 상담 |

---

## 상담의 중요성

| 포인트 | 설명 |
|--------|------|
| 첫 인상 | 상담이 환자의 첫 인상을 결정 |
| 전환율 | 좋은 상담이 예약/방문으로 이어짐 |
| 재방문 | 친절한 응대가 재방문 유도 |

---

## 상담 팁

좋은 상담을 위해 빠른 응대(채팅은 5분 이내 답변), 친절하고 전문적인 말투, 정확한 정보 제공, 예약/방문 유도를 해주세요.
