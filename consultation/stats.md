# 상담 통계

<figure><img src="../.gitbook/assets/consultation-stats.png" alt="상담 통계"><figcaption>상담 통계 화면</figcaption></figure>

## 상담 통계란?

상담 통계에서는 전화 상담, 채팅 상담의 데이터를 확인하고 분석할 수 있습니다.

---

## 확인 가능한 통계

| 지표 | 설명 |
|------|------|
| 총 상담 수 | 전화 + 채팅 상담 합계 |
| 앱전화 수 | 앱을 통한 전화 연결 수 |
| 채팅 상담 수 | 채팅 상담 수 |
| 평균 응답 시간 | 채팅 첫 응답까지 걸린 시간 |
| 이벤트별 상담 | 이벤트별 상담 수 |
| 시간대별 상담 | 시간대별 상담 분포 |

---

## 통계 활용

상담 통계를 분석하면 어떤 이벤트에 관심이 많은지, 어떤 시간대에 상담이 많은지, 응답 시간 개선이 필요한지 파악할 수 있습니다.
