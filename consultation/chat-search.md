## 🔍 검색 방법

<details>
<summary><strong>기본 검색</strong></summary>

**검색 방법:**
1. 채팅 목록 상단 검색창 클릭
2. 검색어 입력
3. 검색 결과 확인

**검색 가능 항목:**
- 고객 이름/닉네임
- 상담 내용 키워드
- 시술명
- 예약 관련 내용

</details>

<details>
<summary><strong>필터 검색</strong></summary>

**필터 옵션:**
- 기간 (오늘/7일/30일/사용자지정)
- 상담 상태 (진행중/완료)
- 담당자

</details>
