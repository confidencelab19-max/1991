# 🕐 운영 시간 설정

{% hint style="info" %}
**정확한 운영 시간 설정은 필수입니다!**

잘못된 운영 시간은 고객 불만의 원인이 됩니다.
{% endhint %}

---

## ⏰ 운영 시간 설정

<details>
<summary><strong>요일별 진료 시간</strong></summary>

**설정 경로:**
파트너센터 → 병원관리 → 운영시간

**설정 항목:**
| 요일 | 진료시작 | 진료종료 | 점심시간 |
|------|----------|----------|----------|
| 월요일 | 09:00 | 18:00 | 12:00-13:00 |
| 화요일 | 09:00 | 18:00 | 12:00-13:00 |
| 수요일 | 09:00 | 18:00 | 12:00-13:00 |
| 목요일 | 09:00 | 18:00 | 12:00-13:00 |
| 금요일 | 09:00 | 18:00 | 12:00-13:00 |
| 토요일 | 09:00 | 14:00 | - |
| 일요일 | 휴진 | - | - |

</details>

<details>
<summary><strong>야간 진료 설정</strong></summary>

야간 진료를 운영하는 경우 설정하세요.

**야간 진료 기준:**
- 평일: 18:00 이후 진료
- 주말: 별도 설정

**설정 예시:**
월~목: 09:00-21:00 (야간 진료) 금: 09:00-18:00 토: 09:00-14:00 일: 휴진
**앱 노출:**
야간 진료 설정 시 "야간진료" 배지가 자동으로 표시됩니다.

</details>

<details>
<summary><strong>점심 시간 설정</strong></summary>

점심 시간에는 예약이 자동으로 차단됩니다.

**설정 옵션:**
- 점심시간 있음: 시작-종료 시간 입력
- 점심시간 없음: 연속 진료

**주의사항:**
- 점심시간에도 전화 상담은 가능하도록 안내 권장
- 점심시간 변경 시 기존 예약 확인 필수

</details>

---

## 📅 예외 시간 설정

<details>
<summary><strong>특정 요일 다른 시간 운영</strong></summary>

특정 요일만 다른 시간으로 운영할 수 있습니다.

**예시: 수요일 오후 휴진**
월,화,목,금: 09:00-18:00 수: 09:00-13:00 (오후 휴진) 토: 09:00-14:00
</details>

<details>
<summary><strong>격주/특정 주 설정</strong></summary>

격주 또는 특정 주만 다른 시간으로 운영할 수 있습니다.

**예시: 격주 토요일 휴진**
매주 토요일: 09:00-14:00 단, 둘째/넷째 토요일: 휴진
**설정 방법:**
휴진일 관리에서 반복 설정으로 등록

➡️ [휴진일 관리 보기](holidays.md)

</details>
