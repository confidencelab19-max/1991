## 알아두면 좋은 정보

{% hint style="warning" %}
**병원 정보 변경 시 주의**

병원명, 주소 등 주요 정보 변경 시 검토 후 반영됩니다. 변경 완료까지 1-2 영업일이 소요될 수 있습니다.
{% endhint %}

{% hint style="info" %}
**의사 정보는 사실상 필수**

이벤트 등록 전 최소 1명의 의사 정보가 등록되어 있어야 합니다.
{% endhint %}
