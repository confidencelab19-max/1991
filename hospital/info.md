# 🏢 병원 정보 관리

{% hint style="info" %}
**병원 정보는 고객 신뢰의 첫걸음입니다.**

정확하고 매력적인 정보로 고객을 유치하세요.
{% endhint %}

---

## 📋 기본 정보 설정

<details>
<summary><strong>병원명</strong></summary>

**입력 규칙:**
- 의료기관 개설허가증 상의 정식 명칭 사용
- 띄어쓰기 정확히 입력
- 특수문자 사용 불가

**예시:**
- ✅ 자신있나비뇨의학과의원
- ❌ 자신있나 비뇨기과 (띄어쓰기 오류)
- ❌ #자신있나비뇨기과# (특수문자)

</details>

<details>
<summary><strong>병원 소개</strong></summary>

**작성 가이드:**
- 500자 이상 권장
- 병원의 특장점 중심으로 작성
- 객관적이고 신뢰감 있는 표현 사용

**포함 권장 내용:**
- 설립 연도 및 연혁
- 대표 시술 분야
- 의료진 특징
- 시설 및 장비
- 수상 이력 (있는 경우)

**작성 예시:**
자신있나비뇨의학과는 2015년 개원 이래 남성 건강 전문 의료기관으로 10년간 10만 건 이상의 시술 경험을 보유하고 있습니다.

비뇨의학과 전문의 3인이 상주하며, 최신 의료장비와 무균 수술실을 갖추고 있습니다.

정관수술, 포경수술, 남성확대술 등 다양한 시술에서 높은 만족도를 기록하고 있습니다.
</details>

<details>
<summary><strong>병원 사진</strong></summary>

**필수 사진 (최소 5장):**
| 구분 | 설명 | 권장 구도 |
|------|------|----------|
| 외관 | 병원 간판이 보이는 전경 | 정면 또는 45도 각도 |
| 로비 | 대기실 전경 | 넓어 보이는 각도 |
| 진료실 | 진료 공간 | 깔끔한 구도 |
| 상담실 | 상담 공간 | 프라이버시 강조 |
| 수술실 | 시설 현황 | 청결함 강조 |

**사진 규격:**
- 권장 해상도: 1200 x 800px
- 파일 형식: JPG, PNG
- 파일 크기: 5MB 이하

**주의사항:**
- 환자가 식별되는 사진 불가
- 타 병원 사진 도용 불가
- 과도한 보정 불가

</details>

## 📍 위치 정보

<details>
<summary><strong>주소 입력</strong></summary>

**입력 순서:**
1. 우편번호 검색
2. 기본 주소 자동 입력
3. 상세 주소 직접 입력 (층수, 호수)

**예시:**
- 기본 주소: 서울시 강남구 테헤란로 123
- 상세 주소: OO빌딩 5층 501호

</details>
