# 📅 휴진일 관리

{% hint style="warning" %}
**휴진일 설정은 최소 7일 전에 해주세요!**

기존 예약이 있는 날짜는 휴진 설정이 제한됩니다.
{% endhint %}

---

## 🗓️ 휴진일 설정

<details>
<summary><strong>단일 휴진일 등록</strong></summary>

특정 날짜만 휴진하는 경우

**설정 경로:**
파트너센터 → 병원관리 → 휴진일 관리 → 휴진일 추가

**입력 항목:**
- 휴진 날짜: 달력에서 선택
- 휴진 사유: (선택) 명절, 개인사유, 학회 등
- 고객 안내 메시지: (선택) 앱에 표시될 안내 문구

</details>

<details>
<summary><strong>연속 휴진 등록</strong></summary>

여러 날 연속 휴진하는 경우 (예: 명절, 휴가)

**설정 방법:**
1. 휴진일 추가 클릭
2. "연속 휴진" 선택
3. 시작일 ~ 종료일 선택
4. 사유 입력

**예시:**
설 연휴: 2026.01.28 ~ 2026.01.30 (3일간) 여름휴가: 2026.08.10 ~ 2026.08.14 (5일간)

</details>

<details>
<summary><strong>반복 휴진 등록</strong></summary>

정기적으로 반복되는 휴진일

**반복 옵션:**
- 매주: 매주 특정 요일 (예: 매주 일요일)
- 격주: 2주에 1회 (예: 격주 토요일)
- 매월: 매월 특정 일 (예: 매월 첫째 월요일)

**예시:**
매주 일요일: 휴진 격주 토요일: 휴진 (둘째, 넷째) 매월 첫째 월요일: 휴진
</details>

---

## ⚠️ 휴진일 설정 시 주의사항

{% hint style="danger" %}
**기존 예약이 있는 날짜는 주의하세요!**
{% endhint %}

<details>
<summary><strong>예약이 있는 날 휴진 설정 시</strong></summary>

이미 예약이 있는 날짜에 휴진을 설정하면:

1. **휴진 설정 불가**: 기본적으로 예약이 있으면 휴진 설정이 차단됩니다.

2. **강제 휴진 설정 시**:
   - 해당 예약 고객에게 자동 알림 발송
   - 예약 변경/취소 처리 필요
   - 고객 불만 발생 가능

**권장 절차:**
1. 휴진일 최소 7일 전에 설정
2. 기존 예약 고객에게 개별 연락
3. 예약 일정 변경 또는 취소 처리
4. 휴진일 설정

</details>

<details>
<summary><strong>공휴일 자동 휴진</strong></summary>

**자동 휴진 설정:**
파트너센터 → 병원관리 → 휴진일 관리 → 공휴일 설정

**옵션:**
- 모든 공휴일 자동 휴진
- 일부 공휴일만 휴진
- 공휴일도 정상 진료

**대체공휴일:**
대체공휴일도 공휴일로 자동 인식됩니다.

</details>
