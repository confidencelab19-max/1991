# 시작 가이드

<figure><img src="../.gitbook/assets/getting-started-steps.png" alt="시작 가이드"><figcaption>3단계로 기본 설정을 완료할 수 있습니다</figcaption></figure>

계정을 발급받으셨나요? 이제 파트너센터를 설정하고 첫 이벤트를 등록해보세요. 이 가이드를 따라하면 10분 안에 모든 기본 설정을 완료할 수 있습니다.

---

## 설정 체크리스트

| 순서 | 항목 | 예상 시간 | 필수 여부 |
|------|------|-----------|-----------|
| 1 | 병원 정보 입력 | 3분 | 필수 |
| 2 | 의사 정보 등록 | 3분 | 필수 |
| 3 | 이벤트 등록 | 4분 | 필수 |
| 4 | 예약 설정 | 3분 | 선택 |
| 5 | 알림 설정 | 1분 | 선택 |

---

## Step 1: 병원 정보 입력

가장 먼저 병원 기본 정보를 입력합니다.

<figure><img src="../.gitbook/assets/hospital-info-form.png" alt="병원 정보 입력"><figcaption>병원 정보 입력 화면</figcaption></figure>

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 병원 관리를 클릭하고, 병원 정보를 클릭합니다.

### 입력 항목

| 항목 | 입력 내용 | 필수 |
|------|-----------|------|
| 병원명 | 정식 병원명 입력 | 필수 |
| 대표 전화번호 | 환자 연락용 대표번호 | 필수 |
| 주소 | 병원 주소 검색 후 선택 | 필수 |
| 상세 주소 | 층수, 호수 등 상세 주소 | 필수 |
| 운영 시간 | 요일별 운영 시간 | 필수 |
| 병원 소개 | 병원 특징, 장점 소개 | 권장 |
| 병원 사진 | 외관, 내부, 시설 사진 | 권장 |

모든 정보 입력 후 저장 버튼을 클릭합니다.

---

## Step 2: 의사 정보 등록

병원에서 진료하는 의사 정보를 등록합니다.

<figure><img src="../.gitbook/assets/doctor-form.png" alt="의사 정보 등록"><figcaption>의사 정보 입력 화면</figcaption></figure>

### 접속 방법

좌측 메뉴에서 병원 관리를 클릭하고, 의사 정보를 클릭한 후 의사 추가 버튼을 클릭합니다.

### 입력 항목

| 항목 | 입력 내용 | 필수 |
|------|-----------|------|
| 의사명 | 의사 성명 | 필수 |
| 프로필 사진 | 정면 프로필 사진 | 필수 |
| 전문 분야 | 전문 수술 분야 선택 | 필수 |
| 경력 | 주요 경력 사항 | 필수 |
| 학력 | 학력 사항 | 선택 |
| 의사 소개 | 간단한 자기소개 | 권장 |

모든 정보 입력 후 저장 버튼을 클릭합니다. 의사가 여러 명인 경우 의사 추가 버튼을 눌러 추가 등록할 수 있습니다.

---

## Step 3: 이벤트 등록

환자에게 노출할 수술 이벤트를 등록합니다.

<figure><img src="../.gitbook/assets/event-create-form.png" alt="이벤트 등록"><figcaption>이벤트 등록 화면</figcaption></figure>

### 접속 방법

좌측 메뉴에서 이벤트 관리를 클릭하고, 이벤트 설정을 클릭한 후 이벤트 등록 버튼을 클릭합니다.

### 입력 항목

| 항목 | 입력 내용 | 필수 |
|------|-----------|------|
| 수술 종류 | 수술 카테고리 선택 | 필수 |
| 이벤트명 | 이벤트 제목 | 필수 |
| 가격 | 수술 가격 (VAT 포함) | 필수 |
| 이벤트 설명 | 상세 설명 | 필수 |
| 이벤트 이미지 | 대표 이미지 | 필수 |
| 담당 의사 | 담당 의사 선택 | 필수 |

모든 정보 입력 후 검수 요청 버튼을 클릭합니다. 검수는 평일 기준 1-2 영업일이 소요됩니다.

---

## Step 4: 예약 설정 (선택)

앱예약 기능을 사용하시려면 예약 설정을 완료해주세요.

<figure><img src="../.gitbook/assets/reservation-schedule-setting.png" alt="예약 설정"><figcaption>예약 설정 화면</figcaption></figure>

### 설정 항목

| 항목 | 설정 내용 |
|------|-----------|
| 예약 가능 시간 | 예약 받을 시간대 설정 |
| 예약 간격 | 예약 시간 간격 (30분/1시간) |
| 예약 확정 방식 | 자동 확정 / 수동 확정 선택 |

자세한 내용은 [예약 관리](../reservation/README.md) 가이드를 참고하세요.

---

## Step 5: 알림 설정 (선택)

새 예약, 상담, 리뷰 알림을 받으시려면 알림 설정을 완료해주세요.

### 설정 방법

우측 상단 설정 아이콘을 클릭하고, 알림 설정을 클릭합니다.

### 알림 종류

| 알림 종류 | 설명 |
|-----------|------|
| 앱 푸시 | 모바일 앱으로 알림 |
| 이메일 | 이메일로 알림 |
| 알림톡 | 카카오 알림톡으로 알림 |

---

## 설정 완료

축하합니다. 모든 기본 설정이 완료되었습니다. 이제 이벤트 검수가 완료되면 환자들이 병원을 볼 수 있습니다.
