## 빠른 시작 (10분 완성)

파트너센터를 처음 사용하신다면, 아래 순서대로 설정해보세요.

<figure><img src="../.gitbook/assets/getting-started-steps.png" alt="시작 단계"><figcaption>3단계로 기본 설정을 완료할 수 있습니다</figcaption></figure>

| 단계 | 내용 | 소요 시간 |
|------|------|-----------|
| Step 1 | 병원 정보 입력 | 3분 |
| Step 2 | 의사 정보 등록 | 3분 |
| Step 3 | 첫 이벤트 등록 | 4분 |

자세한 설정 방법은 [시작 가이드](getting-started.md)를 참고하세요.

## 알아두면 좋은 정보

### 검수 시간 안내

이벤트 등록 후 검수까지 평일 기준 1-2 영업일이 소요됩니다. 주말과 공휴일에는 검수가 진행되지 않습니다.

### 필수 정보 안내

병원 정보와 의사 정보가 모두 입력되어야 이벤트 등록이 가능합니다. 먼저 기본 정보를 입력해주세요.
