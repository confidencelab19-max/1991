# 📌 시작하기

{% hint style="info" %}
**처음 오셨나요?**

아래 순서대로 진행하시면 빠르게 파트너센터를 시작할 수 있습니다.
{% endhint %}

---

## 🚀 빠른 시작 가이드

<details>
<summary><strong>Step 1: 서비스 이해하기</strong></summary>

자신있나는 비뇨기과 전문 의료 플랫폼입니다.

**주요 특징:**
- 비뇨기과/남성의학 특화 플랫폼
- 월 30만+ 활성 사용자
- 평균 전환율 업계 최고 수준

➡️ [서비스 소개 자세히 보기](about.md)

</details>

<details>
<summary><strong>Step 2: 입점 신청하기</strong></summary>

파트너 병원 입점 신청 절차입니다.

**필요 서류:**
- 사업자등록증
- 의료기관 개설허가증
- 전문의 자격증 사본
- 대표자 신분증 사본

**신청 방법:**
1. 파트너센터 접속
2. 입점 신청서 작성
3. 필요 서류 업로드
4. 심사 대기 (영업일 3-5일)

➡️ [입점 신청 바로가기](apply.md)

</details>

<details>
<summary><strong>Step 3: 심사 통과하기</strong></summary>

입점 심사 기준을 확인하세요.

**심사 항목:**
- 의료기관 적법성 확인
- 전문의 자격 확인
- 시설 및 장비 확인
- 서비스 품질 평가

**심사 기간:** 영업일 기준 3-5일

➡️ [심사 기준 상세 보기](approval.md)

</details>

<details>
<summary><strong>Step 4: 초기 설정하기</strong></summary>

심사 통과 후 초기 설정을 완료하세요.

**필수 설정:**
1. 병원 기본 정보 입력
2. 운영 시간 설정
3. 의사 정보 등록
4. 첫 이벤트 등록

➡️ [시작 가이드 보기](getting-started.md)

</details>

<details>
<summary><strong>Step 5: 대시보드 활용하기</strong></summary>

대시보드에서 모든 데이터를 확인하세요.

**확인 가능 항목:**
- 오늘의 예약 현황
- 이벤트 성과
- 리뷰 알림
- 정산 현황

➡️ [대시보드 가이드 보기](dashboard.md)

</details>
