# 입점 신청하기

<figure><img src="../.gitbook/assets/apply-process.png" alt="입점 신청 프로세스"><figcaption>입점 신청부터 계정 발급까지의 프로세스</figcaption></figure>

## 입점 신청 안내

자신있나 파트너 병원으로 입점하시면, 비뇨기과 수술을 고민하는 환자들에게 병원을 알리고 새로운 환자를 만날 수 있습니다.

---

## 입점 자격

아래 조건을 충족하는 병원만 입점 신청이 가능합니다.

| 조건 | 상세 |
|------|------|
| 의료기관 종별 | 의원, 병원, 종합병원 |
| 진료과목 | 비뇨의학과(비뇨기과) 필수 |
| 전문의 | 비뇨의학과 전문의 1인 이상 |
| 사업자등록 | 유효한 사업자등록증 보유 |
| 의료기관 개설허가 | 유효한 의료기관 개설허가증 보유 |

---

## 신청 절차

### Step 1: 입점 신청서 제출

아래 정보를 준비하여 partner@jasininna.com으로 제출해주세요.

필수 서류는 다음과 같습니다.

| 서류 | 설명 |
|------|------|
| 사업자등록증 사본 | 발급 3개월 이내 |
| 의료기관 개설허가증 사본 | 유효한 허가증 |
| 전문의 자격증 사본 | 비뇨의학과 전문의 |
| 대표자 신분증 사본 | 주민번호 뒷자리 마스킹 |

### Step 2: 서류 검토

제출된 서류를 검토합니다. 영업일 기준 3-5일이 소요됩니다.

### Step 3: 계약 체결

서류 검토 통과 시 담당자가 연락드려 계약을 진행합니다.

### Step 4: 계정 발급

계약 완료 후 파트너센터 계정을 발급해 드립니다.

### Step 5: 병원 정보 등록

파트너센터에 로그인하여 병원 정보를 등록합니다.

신규 입점 병원에게 다음 혜택을 제공합니다.

| 혜택 | 내용 |
|------|------|
| 부가광고 할인 | 입점 후 첫 달 50% 할인 |
| 전담 매니저 | 전담 매니저 배정 |
| 사용 교육 | 파트너센터 사용 교육 제공 |

## 자주 묻는 질문

### Q. 입점 비용이 있나요?

입점 자체는 무료입니다. 단, 이벤트 등록 후 상담이 발생하면 건당 수수료가 발생합니다.

### Q. 개인병원도 입점 가능한가요?

네, 비뇨의학과 전문의가 있는 개인 의원도 입점 가능합니다.

### Q. 여러 지점이 있는 경우 어떻게 하나요?

각 지점별로 별도 입점 신청이 필요합니다. 단, 본원에서 일괄 관리도 가능합니다.
