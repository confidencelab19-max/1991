# 📊 대시보드 가이드

{% hint style="info" %}
**대시보드에서 모든 성과를 한눈에!**

실시간 예약, 이벤트 성과, 리뷰 현황을 확인하세요.
{% endhint %}

---

## 🖥️ 대시보드 구성

<details>
<summary><strong>1. 오늘의 현황</strong></summary>

**표시 항목:**
- 오늘 예약 건수
- 대기 중인 예약
- 확정된 예약
- 취소된 예약

**활용 팁:**
- 매일 아침 확인하여 당일 일정 파악
- 미확정 예약은 빠르게 처리

</details>

<details>
<summary><strong>2. 이벤트 성과</strong></summary>

**표시 항목:**
- 이벤트별 조회수
- 상담 신청 수
- 예약 전환율
- 매출 현황

**활용 팁:**
- 전환율 낮은 이벤트 개선
- 인기 이벤트 분석

</details>

<details>
<summary><strong>3. 리뷰 알림</strong></summary>

**표시 항목:**
- 새로운 리뷰
- 미답변 리뷰
- 평균 평점 변화
- 리뷰 키워드

**활용 팁:**
- 24시간 내 답변 권장
- 부정 리뷰 신속 대응

</details>

<details>
<summary><strong>4. 정산 현황</strong></summary>

**표시 항목:**
- 이번 달 매출
- 수수료 현황
- 정산 예정액
- 정산 완료액

**활용 팁:**
- 매월 정산 명세서 확인
- 이상 내역 즉시 문의

</details>

## 📈 데이터 분석 기능

<details>
<summary><strong>기간별 통계</strong></summary>

**조회 기간:**
- 오늘
- 최근 7일
- 최근 30일
- 사용자 지정

**분석 항목:**
- 방문자 추이
- 예약 추이
- 매출 추이
- 전환율 추이

</details>

- 이벤트별 전환율
- 이벤트별 매출
- 시술 카테고리별 성과

</details>

<details>

- 지역별 분포
- 유입 경로
- 재방문율

</details>
