# 파트너센터란

<figure><img src="../.gitbook/assets/partner-center-overview.png" alt="파트너센터 화면"><figcaption>자신있나 파트너센터 메인 화면</figcaption></figure>

## 파트너센터 소개

파트너센터는 자신있나 파트너 병원을 위한 전용 관리 시스템입니다. 병원 정보 관리부터 이벤트 등록, 예약 관리, 상담 관리까지 모든 업무를 파트너센터에서 처리할 수 있습니다.

---

## 접속 방법

### PC 접속

웹 브라우저에서 partner.jasininna.com에 접속한 후 발급받은 계정으로 로그인합니다.

### 모바일 접속

모바일 브라우저에서도 동일하게 partner.jasininna.com에 접속하여 사용할 수 있습니다.

권장 브라우저는 Chrome, Safari, Edge 최신 버전입니다.

## 메뉴 구성

<figure><img src="../.gitbook/assets/menu-structure.png" alt="메뉴 구조"><figcaption>파트너센터 메뉴 구조</figcaption></figure>

## 대시보드 상세

<figure><img src="../.gitbook/assets/dashboard-detail.png" alt="대시보드 상세"><figcaption>대시보드 화면 상세</figcaption></figure>

로그인하면 가장 먼저 대시보드가 표시됩니다.

### 주요 지표

| 지표 | 설명 |
|------|------|
| 상담시작 | 앱에서 상담을 시작한 환자 수 |
| 앱전화 | 앱을 통해 전화가 연결된 수 |
| 이벤트시작 | 이벤트 페이지에서 상담을 시작한 수 |
| 병원 후기 | 새로 등록된 후기 수 |
| 병원리뷰 | 병원 평균 별점 |
| 페이지 조회수 | 병원 페이지 방문자 수 |

### 최근 활동

대시보드 하단에는 최근 등록된 예약, 상담 내역, 후기가 표시됩니다.

## 계정 관리

### 계정 종류

| 계정 종류 | 권한 |
|-----------|------|
| 관리자 | 모든 기능 사용 가능 |
| 일반 | 조회 및 일부 기능만 사용 가능 |

추가 계정이 필요하시면 partner@jasininna.com으로 문의해주세요.

---

## 알림 설정

새로운 예약, 상담, 리뷰가 등록되면 알림을 받을 수 있습니다.

| 알림 종류 | 설명 |
|-----------|------|
| 앱 푸시 | 모바일 앱으로 실시간 알림 |
| 이메일 | 등록된 이메일로 알림 발송 |
| 알림톡 | 카카오 알림톡으로 알림 발송 |

알림 설정은 파트너센터 우측 상단 설정 아이콘에서 변경할 수 있습니다.
