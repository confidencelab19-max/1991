# 2026년 1월 업데이트

2026년 1월, 선정된 파트너 병원들과 함께 베타 테스트를 성공적으로 완료했습니다.

## 개선된 기능

베타 테스트 피드백을 반영하여 아래 기능들이 개선되었습니다.

### 예약 관리 개선

예약 목록 로딩 속도가 50% 향상되었습니다. 예약 상태 필터 기능이 추가되었고, 날짜 범위 선택 기능이 개선되었습니다.

### 이벤트 등록 개선

이미지 업로드 용량 제한이 5MB에서 10MB로 완화되었습니다. 임시저장 기능이 추가되었고, 검수 반려 사유가 더 상세하게 안내됩니다.

### 상담 관리 개선

채팅 알림의 실시간성이 개선되었습니다. 상담 내역 검색 기능이 추가되었고, 자주 쓰는 답변 템플릿 기능이 추가되었습니다.
