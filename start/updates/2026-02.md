## 파트너센터 주요 기능

### 대시보드

<figure><img src="../../.gitbook/assets/dashboard-preview.png" alt="대시보드"><figcaption>실시간으로 주요 지표를 확인할 수 있습니다</figcaption></figure>

대시보드에서 확인 가능한 지표는 다음과 같습니다. 상담시작 수, 앱전화 연결 수, 이벤트시작 수, 병원 후기 수, 병원 평균 별점, 페이지 조회수를 한눈에 볼 수 있습니다.

### 알림 기능

새로운 예약, 상담, 리뷰가 등록되면 실시간으로 알림을 받을 수 있습니다. 앱 푸시 알림, 이메일 알림, 카카오 알림톡 중 원하는 채널을 선택할 수 있습니다.
