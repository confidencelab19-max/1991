## 🔔 2026.02.06 | 채팅 자동응답 메시지 발송 방식 업데이트

{% hint style="info" %}
자동응답 메시지 발송 방식이 새롭게 개선되었습니다.
{% endhint %}

<details>
<summary><strong>변경 사항 상세보기</strong></summary>

**기존 방식:**
- 고객 메시지 수신 후 일정 시간 경과 후 발송
- 영업시간 내/외 동일한 메시지 발송

**변경 방식:**
- 고객 첫 메시지 수신 시 **즉시** 자동응답 발송
- **영업시간 내** 메시지와 **영업시간 외** 메시지 별도 설정
- 다국어 자동응답 지원 (한국어, 영어, 중국어, 일본어)

**설정 경로:**
1. 파트너센터 접속
2. 상담관리 메뉴 클릭
3. 자동응답 설정 선택
4. 영업시간 내/외 메시지 각각 설정

**권장 메시지 예시:**

영업시간 내:안녕하세요! 자신있나 OOO병원입니다. 상담원이 곧 응대해드리겠습니다. 잠시만 기다려주세요. 😊

영업시간 외:안녕하세요! 자신있나 OOO병원입니다. 현재 영업시간이 아닙니다. 영업시간: 평일 09:00-18:00 남겨주신 문의는 익일 순차적으로 답변드리겠습니다.

</details>

## 🆕 2026.02.02 | 위챗 미니프로그램 가이드 오픈

{% hint style="success" %}
중국 고객과 만나는 새로운 채널이 열렸습니다!
{% endhint %}

<details>
<summary><strong>위챗 미니프로그램 상세보기</strong></summary>

**서비스 개요:**
위챗(WeChat)은 중국 최대 메신저 앱으로, 월간 활성 사용자 13억 명 이상을 보유하고 있습니다. 자신있나 위챗 미니프로그램을 통해 중국 고객에게 병원을 노출할 수 있습니다.

**주요 기능:**
- 중국어 이벤트 자동 연동
- 위챗페이(WeChat Pay) 결제 지원
- 중국 고객 전용 채팅 상담
- 실시간 통역 지원 (추가 서비스)

**이용 조건:**
- 중국어 이벤트 1개 이상 등록
- 중국어 자동응답 메시지 설정
- 위챗 운영 정책 동의

**유의사항:**
- 위챗 유입 고객은 별도 과금 기준 적용
- 중국 고객 예약 시 통역 서비스 제공 (유료)
- 결제 시 위안화(CNY) → 원화(KRW) 자동 환산

➡️ [위챗 상세 가이드 보기](wechat.md)

</details>
