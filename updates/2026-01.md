## 💬 2026.01.19 | 채팅화면 검색 기능 업데이트

<details>
<summary><strong>상세 기능 보기</strong></summary>

고객과의 채팅 히스토리를 빠르게 찾을 수 있는 검색 기능이 추가되었습니다.

**검색 가능 항목:**
- 고객 이름/닉네임
- 상담 내용 키워드
- 예약/이벤트 관련 대화
- 기간별 검색

**사용 방법:**
1. 채팅 목록 화면 상단 검색창 클릭
2. 검색어 입력
3. 필터 적용 (선택)
4. 검색 결과에서 대화 선택

</details>
