# 위챗 미니프로그램 가이드

{% hint style="info" %}
**중국 고객과 만나는 새로운 채널, 위챗!**

위챗 미니프로그램을 통해 중국 의료관광 고객을 유치하세요.
{% endhint %}

---

## 📱 위챗 미니프로그램이란?

<details>
<summary><strong>서비스 개요</strong></summary>

위챗(WeChat)은 중국 최대 메신저 앱으로, 월간 활성 사용자 13억 명 이상을 보유하고 있습니다.

**미니프로그램 특징:**
- 앱 설치 없이 위챗 내에서 바로 이용
- 중국 본토 사용자 접근 가능
- 위챗페이 결제 연동

**자신있나 위챗 미니프로그램:**
- 중국어 이벤트 자동 노출
- 중국 고객 전용 채팅
- 위챗페이 선결제 지원

</details>

---

## 🚀 시작하기

<details>
<summary><strong>Step 1: 중국어 이벤트 등록</strong></summary>

위챗 미니프로그램에 노출되려면 중국어 이벤트가 필요합니다.

**등록 방법:**
1. 파트너센터 → 이벤트관리 → 이벤트등록
2. 언어 선택: 중국어(简体/繁體)
3. 이벤트 정보 중국어로 입력
4. 검수 요청

**입력 항목 (중국어):**
- 시술명
- 시술 설명
- 주의사항
- 이벤트 이미지 (중국어 텍스트 포함)

</details>

<details>
<summary><strong>Step 2: 중국어 자동응답 설정</strong></summary>

중국 고객 채팅을 위한 자동응답을 설정하세요.

**설정 경로:**
파트너센터 → 상담관리 → 자동응답 설정 → 중국어

**권장 메시지:**
您好！感谢您咨询自信男科医院。 我们的咨询师将尽快为您服务。 营业时间：周一至周五 09:00-18:00（韩国时间）
</details>

<details>
<summary><strong>Step 3: 위챗 정책 동의</strong></summary>

위챗 미니프로그램 운영을 위해 추가 정책에 동의해주세요.

**동의 항목:**
- 위챗 커뮤니티 가이드라인 준수
- 중국 고객 정보 처리 방침
- 위챗페이 이용약관

**동의 경로:**
파트너센터 → 설정 → 서비스 약관 → 위챗 운영 정책

</details>

## 🌐 통역 서비스

<details>
<summary><strong>통역 서비스 안내</strong></summary>

중국 고객 방문 시 실시간 통역 서비스를 이용할 수 있습니다.

**서비스 유형:**
- 채팅 통역: 실시간 채팅 번역
- 전화 통역: 3자 통화 통역
- 방문 통역: 내원 시 동행 통역

**이용 요금:**
| 서비스 | 요금 |
|--------|------|
| 채팅 통역 | 건당 5,000원 |
| 전화 통역 | 분당 3,000원 |
| 방문 통역 | 시간당 50,000원 |

**신청 방법:**
파트너센터 → 상담관리 → 통역 서비스 신청

</details>
