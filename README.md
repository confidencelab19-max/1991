# 자신있나 파트너센터 가이드

<figure><img src=".gitbook/assets/hero-banner.png" alt="자신있나 파트너센터"><figcaption></figcaption></figure>

자신있나는 비뇨기과 수술을 고민하는 남성들이 정확한 정보를 바탕으로 올바른 결정을 내릴 수 있도록 돕는 플랫폼입니다. 이 가이드는 파트너 병원을 위한 공식 사용 설명서입니다.

## 대시보드 미리보기

<figure><img src=".gitbook/assets/dashboard-preview.png" alt="대시보드 화면"><figcaption>파트너센터 대시보드에서 주요 지표를 한눈에 확인할 수 있습니다</figcaption></figure>

대시보드에서 확인할 수 있는 지표는 다음과 같습니다.

| 지표 | 설명 |
|------|------|
| 상담시작 | 앱에서 상담을 시작한 환자 수 |
| 앱전화 | 앱을 통해 전화 연결된 수 |
| 이벤트시작 | 이벤트 페이지 조회 후 상담 시작 수 |
| 병원 후기 | 새로 등록된 후기 수 |
| 병원리뷰 | 평균 별점 |
| 페이지 조회수 | 병원 페이지 방문 수 |
