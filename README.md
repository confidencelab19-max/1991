# 🏥 자신있나 파트너센터 가이드

<figure><img src=".gitbook/assets/banner.png" alt="자신있나 파트너센터"><figcaption><p>비뇨기과 전문 플랫폼</p></figcaption></figure>

{% hint style="info" %}
**자신있나 파트너센터에 오신 것을 환영합니다!**

이 가이드는 파트너 병원이 자신있나 플랫폼을 효과적으로 활용할 수 있도록 모든 기능과 정책을 상세히 안내합니다. 비뇨기과 전문 플랫폼으로서 남성 건강에 특화된 서비스를 제공합니다.
{% endhint %}

---

## 📢 최신 공지사항

{% hint style="warning" %}
**중요 공지** | 2026년 2월 업데이트 사항을 꼭 확인해주세요!
{% endhint %}

<details>
<summary><strong>🔔 [2026.02.06] 채팅 자동응답 메시지 발송 방식 업데이트</strong></summary>

자동응답 메시지 발송 방식이 새롭게 개선되었습니다.

**변경 사항:**
- 고객 첫 메시지 수신 시 자동응답 즉시 발송
- 영업시간 외 자동응답 메시지 별도 설정 가능
- 다국어 자동응답 지원 (한국어, 영어, 중국어, 일본어)

**적용 일시:** 2026년 2월 6일부터 순차 적용

➡️ [상세 가이드 보기](updates/2026-02.md)

</details>

<details>
<summary><strong>🆕 [2026.02.02] 위챗 미니프로그램 가이드 오픈</strong></summary>

중국 고객과 만나는 새로운 채널, 위챗 미니프로그램이 오픈되었습니다.

**주요 기능:**
- 중국어 이벤트 자동 연동
- 위챗페이 결제 지원
- 중국 고객 전용 채팅 상담

**유의사항:**
- 위챗 고객 유입 시 별도 과금 기준 적용
- 중국어 이벤트 등록 필수

➡️ [위챗 가이드 보기](updates/wechat.md)

</details>

<details>
<summary><strong>🔍 [2026.01.29] 이벤트 승인요청 내역 검색/필터 기능 추가</strong></summary>

이벤트 승인요청 내역을 더 빠르고 쉽게 확인할 수 있습니다.

**새로운 기능:**
- 기간별 검색 (일/주/월/사용자지정)
- 상태별 필터 (대기중/승인/반려/수정요청)
- 시술 카테고리별 필터
- 엑셀 다운로드 기능

➡️ [상세 가이드 보기](event/search-filter.md)

</details>

<details>
<summary><strong>🌏 [2026.01.29] 채팅 자동응답 메시지 국가 확대</strong></summary>

해외 고객 채팅 상담을 위한 자동응답 메시지 대응 국가가 확대되었습니다.

**지원 국가 (기존 4개국 → 12개국):**
- 기존: 한국, 일본, 중국, 미국
- 추가: 태국, 베트남, 인도네시아, 말레이시아, 싱가포르, 대만, 홍콩, 호주

➡️ [다국어 설정 가이드 보기](consultation/multilingual.md)

</details>

<details>
<summary><strong>💬 [2026.01.19] 채팅화면 검색 기능 업데이트</strong></summary>

고객과의 채팅 히스토리를 빠르게 찾을 수 있는 검색 기능이 추가되었습니다.

**검색 가능 항목:**
- 고객 이름/닉네임
- 상담 내용 키워드
- 예약/이벤트 관련 대화
- 기간별 검색

➡️ [채팅 검색 가이드 보기](consultation/chat-search.md)

</details>
