# 🛡️ 악성 리뷰 대응

{% hint style="warning" %}
**악성 리뷰도 전문적으로 대응하세요!**

감정적 대응은 오히려 역효과를 낳습니다.
{% endhint %}

---

## 📋 악성 리뷰 유형

<details>
<summary><strong>1. 정당한 불만 리뷰</strong></summary>

실제 불만족 경험에 기반한 리뷰

**대응 방법:**
- 진심 어린 사과
- 구체적 개선 약속
- 오프라인 상담 제안

**답변 예시:**
OOO님, 불편을 드려 진심으로 사과드립니다. 말씀하신 [구체적 내용]에 대해 내부적으로 즉시 점검하고 개선하겠습니다. 자세한 상담을 원하시면 02-1234-5678로 연락 주시면 담당자가 직접 응대해 드리겠습니다. 불편을 드려 다시 한번 사과드립니다.
</details>

<details>
<summary><strong>2. 허위/비방 리뷰</strong></summary>

사실과 다른 내용이나 악의적 비방

**대응 방법:**
1. 사실관계 정중히 해명
2. 삭제 요청 (플랫폼 신고)
3. 법적 대응 검토

**답변 예시:**
안녕하세요, OOO병원입니다. 해당 리뷰 내용을 확인한 결과, 사실과 다른 부분이 있어 안내드립니다. [사실관계 해명] 오해가 있으시다면 02-1234-5678로 연락 주시면 상세히 안내드리겠습니다.
</details>

<details>
<summary><strong>3. 경쟁업체 의심 리뷰</strong></summary>

방문 이력이 없는 의심 리뷰

**대응 방법:**
1. 정중히 방문 확인 요청
2. 플랫폼에 신고
3. 증거 수집

</details>

---

## 🚨 리뷰 신고 방법

<details>
<summary><strong>신고 절차</strong></summary>

**신고 경로:**
파트너센터 → 리뷰관리 → 해당 리뷰 → 신고

**신고 사유:**
- 허위 사실 유포
- 악의적 비방
- 방문 이력 없음
- 개인정보 노출
- 기타 (직접 입력)

**처리 기간:**
- 접수 후 3-5 영업일 내 검토
- 결과 알림톡 발송

</details>
