# 리뷰 확인하기

<figure><img src="../.gitbook/assets/review-detail.png" alt="리뷰 상세"><figcaption>리뷰 상세 화면</figcaption></figure>

## 리뷰 확인 방법

### 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 리뷰 관리를 클릭합니다.

### 리뷰 목록 화면

| 항목 | 설명 |
|------|------|
| 작성자 | 리뷰 작성 환자 |
| 별점 | 1~5점 별점 |
| 내용 | 리뷰 내용 |
| 작성일 | 리뷰 작성 날짜 |
| 이벤트 | 관련 이벤트 |

## 리뷰 알림

새 리뷰가 등록되면 앱 푸시 알림, 이메일 알림(설정 시)으로 알림을 받습니다.
