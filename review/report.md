# 리뷰 신고하기

## 신고 가능한 리뷰

아래에 해당하는 리뷰는 신고할 수 있습니다.

| 사유 | 설명 |
|------|------|
| 허위 리뷰 | 방문 사실이 없는 허위 리뷰 |
| 욕설/비방 | 욕설이나 심한 비방이 포함된 리뷰 |
| 광고/스팸 | 광고성 내용이나 스팸 리뷰 |
| 개인정보 노출 | 개인정보가 노출된 리뷰 |

---

## 신고 방법

리뷰 목록에서 신고할 리뷰를 클릭하고, 신고 버튼을 클릭합니다. 신고 사유를 선택하고 상세 내용을 입력한 후 제출합니다.

---

## 신고 처리 절차

신고가 접수되면 자신있나 운영팀에서 검토 후 처리 결과를 안내드립니다. 검토에는 1-3 영업일이 소요됩니다.
