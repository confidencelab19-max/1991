# 🎯 타겟팅 설정

{% hint style="info" %}
**정교한 타겟팅으로 광고 효율을 높이세요!**
{% endhint %}

---

## 🎯 타겟팅 옵션

<details>
<summary><strong>요일/시간대 타겟팅</strong></summary>

**설정 옵션:**
- 특정 요일만 노출
- 특정 시간대만 노출

**활용 예시:**
- 주말 집중 노출
- 저녁 시간대 집중 노출

</details>

<details>
<summary><strong>시술명 타겟팅</strong></summary>

**설정 옵션:**
- 특정 시술 검색 시 노출
- 관련 시술 검색 시 노출

</details>

<details>
<summary><strong>인구통계 타겟팅</strong></summary>

**설정 옵션:**
| 항목 | 옵션 |
|------|------|
| 성별 | 남성/여성/전체 |
| 연령 | 20대/30대/40대/50대+ |
| 지역 | 시/도 단위 |

</details>

<details>
<summary><strong>관심사 타겟팅</strong></summary>

**설정 옵션:**
- 고민별 타겟팅
- 최근 검색 기반
- 조회 이력 기반

</details>

<details>
<summary><strong>언어/국가 타겟팅</strong></summary>

**설정 옵션:**
- 한국어 사용자
- 일본어 사용자
- 중국어 사용자
- 영어 사용자

</details>
