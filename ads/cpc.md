# 💰 CPC 광고

{% hint style="info" %}
**CPC (Cost Per Click) - 클릭당 과금 광고**

고객이 이벤트를 클릭할 때만 비용이 발생합니다.
{% endhint %}

---

## 📋 CPC 광고 개요

<details>
<summary><strong>과금 방식</strong></summary>

**원리:**
- 이벤트 노출: 무료
- 고객 클릭 시: 설정한 CPC 단가 과금

**최소 단가:** 100원/클릭
**권장 단가:** 300-500원/클릭 (경쟁도에 따라 상이)

</details>

<details>
<summary><strong>노출 위치</strong></summary>

- 검색 결과 상위
- 카테고리 목록 상위
- 추천 이벤트 영역

</details>

---

## 📊 CPC 광고 성과 분석

<details>
<summary><strong>핵심 지표</strong></summary>

| 지표 | 설명 | 목표 |
|------|------|------|
| 클릭수 | 이벤트 클릭 횟수 | 최대화 |
| 클릭률(CTR) | 노출 대비 클릭 비율 | 3% 이상 |
| 전환율 | 클릭 대비 예약 비율 | 10% 이상 |
| CPA | 예약당 광고비 | 최소화 |

</details>

<details>
<summary><strong>성과 개선 방법</strong></summary>

**클릭률 개선:**
- 매력적인 이벤트 이미지
- 명확한 혜택 제시
- 경쟁력 있는 가격

**전환율 개선:**
- 상세한 시술 설명
- 전후사진 등록
- 신뢰감 있는 병원 정보

</details>

---

## ⚙️ CPC 설정 방법

<details>
<summary><strong>입찰가 설정</strong></summary>

**설정 경로:**
파트너센터 → 광고관리 → CPC 설정

**입찰가 전략:**
| 전략 | 단가 | 효과 |
|------|------|------|
| 상위 노출 | 높음 | 노출↑, 비용↑ |
| 균형 | 중간 | 적정 노출, 적정 비용 |
| 비용 절감 | 낮음 | 노출↓, 비용↓ |

</details>

<details>
<summary><strong>일일 예산 설정</strong></summary>

**예산 설정:**
- 일일 예산 한도 설정
- 예산 소진 시 노출 중지
- 다음 날 자동 재개

**권장 예산:**
- 시작: 일 5만원~
- 성과 확인 후 조절

</details>
