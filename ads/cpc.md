# 🖱️ CPC 광고

## CPC 광고란?

**Cost Per Click** - 클릭당 비용

노출은 무료, 클릭 시에만 과금됩니다.

---

## 💰 과금 구조

| 항목 | 내용 |
|-----|-----|
| 과금 시점 | 클릭 시 |
| 클릭 단가 | 100~500원 |
| 일 예산 | 최소 10,000원 |
| 최소 충전 | 50,000원 |

---

## 📍 노출 위치

- 검색 결과 상단
- 카테고리 상단
- 추천 영역

---

## ⚙️ 설정 방법

### Step 1
**광고 관리** → **CPC 광고**

### Step 2. 캠페인 만들기
| 설정 | 예시 |
|-----|-----|
| 캠페인명 | 정관수술 광고 |
| 일 예산 | 30,000원 |
| 기간 | 상시 |

### Step 3. 광고 그룹
| 설정 | 예시 |
|-----|-----|
| 키워드 | 정관수술, 정관수술 비용 |
| 입찰가 | 200원 |
| 랜딩 | 이벤트 페이지 |

### Step 4
**저장** → 검수 → 시작
