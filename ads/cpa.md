# 🎯 CPA 광고

## CPA 광고란?

**Cost Per Action** - 예약당 비용

예약이 확정될 때만 과금됩니다.

---

## 💰 과금 구조

| 항목 | 내용 |
|-----|-----|
| 과금 시점 | 예약 확정 시 |
| 예약당 단가 | 5,000~20,000원 |
| 노쇼 시 | 과금 없음 |
| 취소 시 | 과금 없음 |

---

## 📍 노출 위치

- 추천 병원
- 검색 결과
- 유사 병원

---

## ⚙️ 설정 방법

### Step 1
**광고 관리** → **CPA 광고**

### Step 2. 설정
| 설정 | 예시 |
|-----|-----|
| 이벤트 | 정관수술 원데이 |
| 예약당 단가 | 10,000원 |
| 월 예산 | 300,000원 |

### Step 3
**저장** → 시작

---

## CPA vs CPC

| 항목 | CPA | CPC |
|-----|-----|-----|
| 과금 | 예약 | 클릭 |
| 리스크 | 낮음 | 중간 |
| 단가 | 높음 | 낮음 |
| 추천 | 효율 중시 | 노출 중시 |
