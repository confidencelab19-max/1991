## 부가광고란?

부가광고는 이벤트를 더 많은 환자에게 노출시킬 수 있는 유료 광고 상품입니다. 부가광고를 이용하면 검색 결과 상위 노출, 메인 페이지 노출 등의 혜택을 받을 수 있습니다.
