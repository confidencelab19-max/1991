## 변경 목록이란?

---

## 확인 가능 정보

| 항목 | 설명 |
|------|------|
| 광고 상품 | 신청한 광고 상품명 |
| 이벤트 | 광고 중인 이벤트 |
| 기간 | 광고 기간 |
| 상태 | 진행중/종료/대기 |
| 변경일 | 변경된 날짜 |

## 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 부가광고 관리를 클릭하고, 부가광고 변경 목록을 클릭합니다.
