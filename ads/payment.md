# 💳 광고비 충전

## 충전 방법

1. **광고 관리** → **광고비 충전**
2. 금액 선택
3. 결제 수단 선택
4. **결제하기**

---

## 💰 충전 금액별 보너스

| 금액 | 보너스 |
|-----|-------|
| 50,000원 | - |
| 100,000원 | +5,000원 |
| 300,000원 | +20,000원 |
| 500,000원 | +50,000원 |
| 1,000,000원 | +120,000원 |

---

## 💳 결제 수단

- 신용카드
- 계좌이체
- 가상계좌

---

## 🧾 세금계산서

- 익월 10일 자동 발행
- **설정** → **사업자 정보**에서 확인
