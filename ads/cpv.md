# 👁️ CPV 광고

{% hint style="info" %}
**CPV (Cost Per View) - 조회당 과금 광고**

고객이 이벤트 상세 페이지를 조회할 때 비용이 발생합니다.
{% endhint %}

---

## 📋 CPV 광고 개요

<details>
<summary><strong>과금 방식</strong></summary>

**원리:**
- 이벤트 목록 노출: 무료
- 상세 페이지 조회 시: 설정한 CPV 단가 과금

**조회 인정 기준:**
- 상세 페이지 3초 이상 체류
- 동일 사용자 1일 1회만 과금

**최소 단가:** 50원/조회

</details>

---

## 📊 CPV 광고 성과 분석

<details>
<summary><strong>핵심 지표</strong></summary>

| 지표 | 설명 | 목표 |
|------|------|------|
| 조회수 | 상세 페이지 조회 횟수 | 최대화 |
| 평균 체류시간 | 페이지 체류 시간 | 30초 이상 |
| 전환율 | 조회 대비 상담/예약 비율 | 5% 이상 |

</details>
