# 📈 광고 성과 분석

{% hint style="info" %}
**데이터 기반으로 광고 효율을 개선하세요!**
{% endhint %}

---

## 📊 성과 대시보드

<details>
<summary><strong>기본 지표</strong></summary>

| 지표 | 설명 |
|------|------|
| 노출수 | 광고가 표시된 횟수 |
| 클릭수 | 광고를 클릭한 횟수 |
| CTR | 클릭률 (클릭/노출) |
| 전환수 | 예약/상담 전환 횟수 |
| CVR | 전환율 (전환/클릭) |
| 비용 | 총 광고 비용 |
| CPA | 전환당 비용 |
| ROAS | 광고비 대비 매출 |

</details>

---

## 📈 성과 개선

<details>
<summary><strong>CTR 개선 방법</strong></summary>

- 매력적인 이벤트 이미지
- 명확한 혜택 제시
- 경쟁력 있는 가격
- 타겟팅 정교화

</details>

<details>
<summary><strong>CVR 개선 방법</strong></summary>

- 상세한 시술 설명
- 전후사진 등록
- 신뢰감 있는 병원 정보
- 리뷰 관리

</details>

<details>
<summary><strong>CPA 최적화</strong></summary>

- 저효율 광고 중지
- 고효율 광고 예산 증액
- A/B 테스트 진행
- 타겟팅 조정

</details>
