# 📈 광고 성과 분석

## 📊 주요 지표

### CPC 광고
| 지표 | 설명 |
|-----|-----|
| 노출 수 | 광고 표시 횟수 |
| 클릭 수 | 클릭 횟수 |
| CTR | 클릭률 |
| CPC | 실제 클릭당 비용 |
| 총 비용 | 기간 내 광고비 |

### CPA 광고
| 지표 | 설명 |
|-----|-----|
| 노출 수 | 광고 표시 횟수 |
| 예약 수 | 확정 예약 수 |
| 전환율 | 예약/노출 |
| CPA | 실제 예약당 비용 |

---

## 📈 리포트 확인

1. **광고 관리** → **리포트**
2. 기간 선택
3. 캠페인 선택
4. **다운로드** (선택)

---

## 💡 성과 개선 팁

### CTR 낮을 때
- 광고 문구 변경
- 이미지 교체
- 키워드 재검토

### 전환율 낮을 때
- 랜딩 페이지 개선
- 가격 경쟁력 확인
- 후기/전후사진 보강
