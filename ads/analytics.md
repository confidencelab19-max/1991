# 광고 성과 분석

<figure><img src="../.gitbook/assets/ads-analytics.png" alt="광고 성과"><figcaption>광고 성과 분석 화면</figcaption></figure>

## 광고 성과 분석이란?

부가광고의 효과를 분석하고 확인할 수 있습니다.

---

## 확인 가능 지표

| 지표 | 설명 |
|------|------|
| 노출수 | 광고가 노출된 횟수 |
| 클릭수 | 광고 클릭 횟수 |
| 클릭률 | 클릭수 / 노출수 |
| 상담 전환 | 광고를 통한 상담 시작 수 |

---

## 접속 방법

파트너센터 로그인 후 좌측 메뉴에서 부가광고 관리를 클릭하고, 해당 광고의 상세 화면에서 성과를 확인합니다.
